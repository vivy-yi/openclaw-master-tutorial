## 一、面向开发者 👨‍💻

### 1.1 技术栈

```
┌─────────────────────────────────────────────────────────────┐
│                     CoPaw 技术栈                             │
├─────────────────────────────────────────────────────────────┤
│  Core                                                       │
│  ├── Python 3.9+ · pip installable                          │
│  ├── LiteLLM (多模型统一接口)                               │
│  └── Asyncio 异步架构                                       │
├─────────────────────────────────────────────────────────────┤
│  Channels                                                   │
│  ├── DingTalk · Feishu · QQ · WeCom                        │
│  ├── Discord · Telegram · iMessage · XiaoYi                │
│  └── Slack · Email · 更多...                               │
├─────────────────────────────────────────────────────────────┤
│  Deployment Options                                         │
│  ├── pip install · 脚本安装                                │
│  ├── Docker 容器化                                         │
│  ├── Desktop App (Beta)                                    │
│  └── ModelScope Studio                                     │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 核心架构

**Skills 系统**:
- 内置 `builtin://cron` 技能支持定时任务
- 自定义技能从 workspace/skills 自动加载
- 支持技能安全扫描

**Memory 系统**:
- 长期记忆存储在用户控制下
- 多工作空间隔离

### 1.3 快速开始

```bash
# 方式1: pip 安装
pip install copaw
copaw init --defaults
copaw app

# 方式2: 一键脚本（无需 Python 环境）
curl -fsSL https://copaw.agentscope.io/install.sh | bash

# 控制台访问
# http://127.0.0.1:8088/
```

### 1.4 本地模型支持

| 框架 | 支持状态 |
|------|----------|
| **llama.cpp** | ✅ 完整支持 |
| **MLX** | ✅ Apple Silicon 优化 |
| **Ollama** | ✅ 本地模型 |

### 1.5 模型提供商

```json
{
  "model": {
    "model_type": "openai",
    "config_name": "gpt-4",
    "api_key": "sk-xxx"
  }
}
```

支持: OpenAI、Anthropic、Azure、DashScope、Ollama 等

## 二、面向产品经理 📊

### 2.1 产品定位

**CoPaw** 是面向个人用户的 **AI 助手**，主打"易安装、易部署、多聊天应用集成"。

### 2.2 核心用户价值

| 用户群体 | 价值主张 |
|----------|----------|
| **个人开发者** | pip 一键安装，快速搭建个人助手 |
| **IM 用户** | 支持 10+ 聊天平台，无缝融入工作流 |
| **隐私敏感用户** | 支持本地模型，数据不出本机 |
| **中文用户** | 钉钉、飞书、QQ、企业微信原生支持 |

### 2.3 差异化特性

1. **中国 IM 生态**: 钉钉、飞书、QQ、企业微信、小爱同学
2. **多部署方式**: pip、脚本、Docker、桌面应用、ModelScope
3. **团队背景**: 阿里巴巴 AgentScope 团队，技术实力保障

### 2.4 使用场景

- **个人助理**: 日程管理、信息查询、任务提醒
- **工作助手**: 对接企业 IM，自动化工作流
- **开发测试**: 快速验证 AI 应用创意
- **学习研究**: 探索多智能体交互

### 2.5 路线图

```
[v0.1.0] 多工作空间 · 技能安全扫描 · 语音转录 · SSE 流式
[规划中] 更多 IM 渠道 · 增强记忆 · 多智能体协作
```

## 三、面向架构师 🏗️

### 3.1 系统架构

**模块化设计**:
```
┌─────────────────────────────────────┐
│           CoPaw Agent               │
├─────────────────────────────────────┤
│  Skills Engine                      │
│  ├── builtin://cron                 │
│  └── custom skills                  │
├─────────────────────────────────────┤
│  Channel Gateway                    │
│  └── 10+ IM adapters                │
├─────────────────────────────────────┤
│  Memory Store                       │
│  └── Local / Remote                 │
└─────────────────────────────────────┘
```

### 3.2 部署模式

| 模式 | 适用场景 | 资源需求 |
|------|----------|----------|
| **pip** | 开发测试 | Python 3.9+ |
| **脚本** | 无 Python 环境 | curl 即可 |
| **Docker** | 生产部署 | Docker Engine |
| **Desktop** | 个人使用 | Windows/Mac/Linux |
| **Studio** | 云端体验 | 浏览器 |

### 3.3 扩展性

- **LiteLLM**: 统一多模型接口，支持 100+ LLM
- **插件化**: Skills 系统支持功能扩展
- **多实例**: 支持多工作空间隔离

### 3.4 安全设计

- **技能安全扫描**: 防止恶意代码执行
- **配置隔离**: 多工作空间数据隔离
- **本地优先**: 可选完全本地运行
