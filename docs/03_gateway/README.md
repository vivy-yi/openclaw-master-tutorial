## 学习目标

完成本章后，你将能够：

- ✅ 理解 Gateway 架构与职责边界
- ✅ 配置 API 端点、认证和限流
- ✅ 掌握控制平面与执行平面的接口契约
- ✅ 理解 WebSocket 连接的生命周期与鉴权机制
- ✅ 掌握事件幂等与重复防护策略
- ✅ 理解设备配对与信任链机制
- ✅ 掌握命令队列与消息处理策略
- ✅ 理解重试策略与渠道行为
- ✅ 监控 Gateway 状态

## 核心概念

```
Gateway = 统一入口 + 路由 + 认证 + 限流 + 控制平面 + 命令队列 + 重试策略
```
