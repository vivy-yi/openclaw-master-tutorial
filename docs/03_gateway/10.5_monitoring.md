## 10.5.1 监控概述

### 监控指标

```
监控指标：

1. 请求指标
   ├── QPS (每秒请求)
   ├── 延迟 (P50/P95/P99)
   └── 错误率

2. 业务指标
   ├── 会话数
   ├── Token 使用
   └── 模型调用

3. 系统指标
   ├── CPU / 内存
   ├── 网络
   └── 磁盘
```

## 10.5.2 日志配置

### 日志级别

```yaml
gateway:
  logging:
    level: info
    levels:
      default: info
      gateway: debug
      agent: warn

    outputs:
      - type: console
      - type: file
        path: ./logs/gateway.log
        rotation:
          max_size: 100MB
          max_files: 10
```

### 日志格式

```yaml
gateway:
  logging:
    format: json
    fields:
      - timestamp
      - level
      - message
      - request_id
      - user_id
      - duration
```

## 10.5.3 指标收集

### Prometheus

```yaml
gateway:
  metrics:
    enabled: true
    provider: prometheus
    port: 9090
    path: /metrics
```

### 指标端点

```
Prometheus 指标：

# HELP gateway_requests_total Total requests
# TYPE gateway_requests_total counter
gateway_requests_total{method="POST",status="200"} 1000

# HELP gateway_request_duration_seconds Request duration
# TYPE gateway_request_duration_seconds histogram
gateway_request_duration_seconds_bucket{le="0.1"} 500
```

## 10.5.4 告警配置

### 告警规则

```yaml
gateway:
  alerts:
    - name: high_error_rate
      condition: error_rate > 0.05
      severity: critical
      channels: [slack, email]

    - name: high_latency
      condition: p99_latency > 5
      severity: warning
      channels: [slack]

    - name: rate_limit
      condition: rate_limit_hits > 100
      severity: info
      channels: [slack]
```
