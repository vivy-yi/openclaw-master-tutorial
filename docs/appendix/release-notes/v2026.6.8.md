## 二、v2026.6.8-beta.2 Highlights（主要更新）

### 2.1 Telegram + WhatsApp 渠道增强

**核心改进**：Telegram 和 WhatsApp 消息投递更加丰富和稳定。

**Telegram 新功能**：
- 支持结构化富文本（表格 tables、列表 lists、可折叠引用 blockquotes）
- 保留 Intentional line breaks（保留换行）
- Prompt-preserving CLI backend delivery
- 废弃 native draft migration
- Safer rich-media boundaries
- Thread 创建 CLI remapping

**WhatsApp 新功能**：
- 支持配置的 ACP bindings（Account-level Configuration Profiles）

## 七、升级建议

### 7.1 适合升级场景

- ✅ 需要 Telegram 富文本（表格/列表/引用）功能
- ✅ 使用 GLM-5.2 或 Claude Haiku 4.5
- ✅ 需要 `/usage` footer renderer 增强
- ✅ 需要 WebChat backscroll 和 iOS reconnect 修复

### 7.2 建议观察场景

- ⚠️ 使用 WhatsApp ACP bindings（新功能，建议观察1-2周再上生产）
