## 二、v2026.6.10-beta.2 Highlights（主要更新）

### 2.1 Auto Fast Mode ⭐ 新功能

**核心改进**：智能切换 Fast Mode，提升短对话效率。

| 特性 | 说明 |
|------|------|
| 短对话自动开启 | 简单查询自动启用 fast mode |
| 长对话自动切换 | 复杂任务切回 normal mode |
| Bounded Fallback | 有保障的降级策略 |

**使用场景**：
- 快速问答 → 自动 fast mode
- 代码调试 → 自动 normal mode
- 混合任务 → 智能识别切换

---

### 2.2 更可靠的模型路由

**核心改进**：多模型协同，提升路由稳定性。

| 功能 | 说明 |
|------|------|
| Zai 模型合成 | 自定义模型组合 |
| GLM 过载 failover | 负载过高自动切换 |
| Native reasoning level 选择 | 原生推理级别优化 |

**优势**：
- 模型选择更准确
- 降级策略更可靠
- 成本优化

---

### 2.3 更安全的 Session/Channel State

**核心改进**：状态管理安全性增强。

| 改进点 | 说明 |
|--------|------|
| Channel 切换重置 | 切换时重置 stale origin fields |
| Cron delivery 保护 | 保持 session 上下文完整性 |

**关联修复**：
- #95866 Gateway 强制重启丢回复（待完全修复）
- #95865 嵌入式 agent run 超时（待完全修复）

---

### 2.4 Trusted Policies 保护

**核心改进**：组合 hook registries 保留 trusted tool policies。

| 场景 | 说明 |
|------|------|
| Tool Policy 保留 | 组合场景下保留安全策略 |
| Hook Registry 保护 | 防止意外覆盖 |
