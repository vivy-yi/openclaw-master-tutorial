## 三、v2026.6.2-beta.1 (2026-06-03)

**发布于**: 2026-06-03
**类型**: Beta（预发布）
**状态**: 🟡 最新Beta

> ⚠️ **注意**: 此版本为预发布版，包含重大安全变更，建议在测试环境验证后再升级生产环境。

### 核心亮点

| 模块 | 变更 | 重要性 |
|------|------|--------|
| **🔐 安全** | **Operator Install Policy 替代 dangerous-code scanner**（重大安全重构） | ⭐⭐⭐⭐⭐ |
| **Channels** | Telegram 重复 transcript 镜像；admin writeback 权限；多渠道安全提升 | ⭐⭐⭐⭐ |
| **UI** | Chat/Control UI/Skill Workshop/Workboard 流式文本可见性改善 | ⭐⭐⭐ |
| **配置恢复** | 拒绝 corrupt shell snapshots；不安全 exec approval precheck；malformed script limits | ⭐⭐⭐⭐ |
| **Android** | Companion shell 体验优化 | ⭐⭐⭐ |

### 重大变更：Operator Install Policy

这是 v2026.6.2-beta.1 最重要的安全变更：

- **替代** `dangerous-code` scanner
- **新机制**: 白名单 + 插件签名验证
- **影响**: 插件安装需要显式授权
- **相关文档**: `16_安全配置/16.11_operator_install_policy.md`

### 升级建议

| 场景 | 建议 |
|------|------|
| **生产环境** | 等待正式版（v2026.6.2）再升级 |
| **安全优先** | 在测试环境验证 Operator Install Policy |
| **开发环境** | 可立即尝鲜，体验新安全机制 |
