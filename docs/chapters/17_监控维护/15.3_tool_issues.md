# 15.3 工具异常排查

## 本节目标

掌握工具调用失败、权限被拒等问题的排查方法。

---

## 15.3.1 排查检查项

### 1. 工具策略检查

是否触发工具治理策略拒绝，或权限不足被拦截：

```bash
# 查看工具策略配置
openclaw tools policy list

# 测试工具调用
openclaw tools test read --path ~/.openclaw
```

**典型报错**：

```
ToolError: execute_bash denied by default policy
Tool 'web_fetch' blocked by security policy
```

**原因**：
- 工具被策略拒绝
- 权限级别不足
- 命令不在白名单

### 2. 参数格式检查

参数格式是否符合预期，是否缺字段或类型不匹配：

**典型报错**：

```
SchemaValidationError: missing required property 'amount'
TypeError: expected string, got number
```

**原因**：
- 缺少必填参数
- 参数类型错误
- 参数值不合法

### 3. 执行环境检查

执行环境是否具备权限与网络访问条件：

```bash
# 查看工具执行日志
openclaw logs --type tool --limit 100
```

**检查项**：
- 沙箱策略是否收紧
- 网络访问是否受限
- 文件权限是否足够

### 4. 上下文检查

工具回执是否成功结构化回注，是否因上下文裁剪而丢失关键信息：

**典型表现**：

```
日志提示：Max context window exceeded, discarding output of tool 'search_web'
```

## 15.3.2 常见工具问题

### 问题 1：文件读取失败

```
错误：File not found: /path/to/file
```

**排查**：
1. 确认文件路径正确
2. 检查文件是否存在
3. 检查读取权限

```bash
ls -la /path/to/file
```

### 问题 2：命令执行被拒

```
错误：Command 'sudo rm -rf /' denied
```

**原因**：
- 命令在黑名单中
- 权限不足

**排查**：
```bash
# 查看工具策略
openclaw tools policy show

# 检查命令是否在白名单
```

### 问题 3：工具超时

```
错误：Tool execution timeout
```

**原因**：
- 执行时间过长
- 网络问题
- 外部服务响应慢

**排查**：
1. 增加超时时间
2. 检查网络连接
3. 优化执行逻辑

### 问题 4：工具输出丢失

```
错误：Tool result not found in context
```

**原因**：
- 上下文被裁剪
- 输出太大被截断
- 结果未正确返回

**排查**：
```bash
# 查看上下文状态
openclaw context status

# 检查裁剪设置
openclaw context config
```

## 15.3.3 诊断命令

```bash
# 1. 列出所有工具
openclaw tools list

# 2. 查看工具详情
openclaw tools info <tool_name>

# 3. 测试工具调用
openclaw tools test <tool_name> --params '{}'

# 4. 查看工具执行日志
openclaw logs --type tool_execution --limit 200

# 5. 查看策略配置
openclaw tools policy show
```
