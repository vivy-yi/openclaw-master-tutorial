## 21.0.3 墨家 Agent 标配 CLI 推荐

> 以下为墨家 Agent 系统推荐安装的 CLI，按工作流分组。

### AI 能力类

```bash
# Claude 编程
brew install claude-code

# 统一 LLM 调用（本地+云端）
pip install llm
llm install modell-latest

# 本地模型
brew install ollama

# 统一网关
pip install litellm
```

### 内容采集类

```bash
# Twitter/X 全功能
brew install --cask xdevplatform/tap/xurl

# 反检测抓取（公众号/知乎）
pip install camoufox

# 视频音频下载
brew install yt-dlp

# 任意URL转Markdown
pip install markitdown
```

### 文档处理类

```bash
# 网站→CLI（B站/小红书等）
npm install -g @jackwener/opencli

# 播客/视频字幕提取
pip install youtube-transcript-api

# 文章正文提取
pip install readabilipy
```

### 浏览器自动化

```bash
# Playwright + MCP
npx playwright install
npm install -g mcp-server-playwright

# AI 浏览器控制
pip install browser-use
```

### 搜索增强

```bash
# Tavily 实时搜索
uvx tavily-search

# 全平台内容采集
# agent-reach（见 ~/.agents/skills/agent-reach/）
```
