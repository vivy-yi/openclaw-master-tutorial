## 21.x3.2 重点平台详解

### 🇨🇳 企业微信（WeCom）—— wecom-cli ⭐1,754

> **官方出品** | 语言：Rust | 安装：`npm install -g @wecom/cli`

**核心能力（7大品类）：**

| 品类 | 能力 |
|------|------|
| 👤 **通讯录** | 获取可见范围成员列表、按姓名/别名搜索 |
| ✅ **待办** | 创建/读取/更新/删除待办，变更处理状态 |
| 🎥 **会议** | 创建预约/取消/更新会议、查询列表与详情 |
| 💬 **消息** | 会话列表、消息记录拉取（文本/图片/文件/语音/视频）、发送文本 |
| 📅 **日程** | 日程增删改查、参与人管理、多成员闲忙查询 |
| 📄 **文档** | 文档创建/读取/编辑 |
| 📊 **智能表格** | 智能表格创建、子表与字段管理、记录增删改查 |

**安装与使用：**
```bash
# 安装 CLI
npm install -g @wecom/cli

# 安装 CLI Skill（推荐）
npx skills add WeComTeam/wecom-cli -y -g

# 配置凭证（交互式，仅需一次）
wecom-cli init

# 常用命令
wecom-cli contact get_userlist '{}'           # 获取成员列表
wecom-cli message send_text '{"content": "hello"}'
wecom-cli meeting create '{"title": "会议"}'
wecom-cli calendar create '{"title": "日程"}'
wecom-cli doc create '{"title": "文档"}'
```

**限制：** 目前仅对 **≤10人企业** 开放使用（企业微信开放平台限制）。

### 🇨🇳 钉钉（DingTalk）—— dingtalk-workspace-cli ⭐1,606

> **钉钉官方出品** | 语言：Go | 安装：官方一键脚本

**核心亮点：**
- 官方跨平台 CLI（macOS/Linux/Windows）
- 统一钉钉全产品线能力
- 内置 Agent Skills，开箱即用
- 自带增量更新（`dws upgrade`）
- 支持 `--dry-run` 预览请求
- 输出格式：`--format table/json/raw`

**安装：**
```bash
# macOS / Linux 一键
curl -fsSL https://raw.githubusercontent.com/DingTalk-Real-AI/dingtalk-workspace-cli/main/scripts/install.sh | sh

# Windows PowerShell
irm https://raw.githubusercontent.com/DingTalk-Real-AI/dingtalk-workspace-cli/main/scripts/install.ps1 | iex

# npm 方式
npm install -g dingtalk-workspace-cli
```

**常用命令：**
```bash
dws upgrade --check          # 检查更新
dws chat list                # 会话列表
dws chat send --to USER "内容"
dws calendar create          # 创建日程
dws meeting create           # 创建会议
dws doc create               # 创建文档
dws search "关键词"          # 搜索
dws upgrade                  # 升级到最新版
```

**安全架构：**
- OAuth device-flow 认证
- 域名白名单
- 最小权限作用域
- 零信任架构

### 🌐 通用桥接工具 —— cc-connect ⭐5,560

> 最通用的消息渠道桥接器，一个工具覆盖所有主流 IM

**支持的平台：**

| 平台 | 方向 |
|------|------|
| **Feishu / Lark** | 收发消息 |
| **DingTalk** | 收发消息 |
| **Slack** | 收发消息 |
| **Telegram** | 收发消息 |
| **Discord** | 收发消息 |
| **LINE** | 收发消息 |
| **WeCom** | 收发消息 |
| **WeChat** | 收发消息 |

```bash
# 安装
npm install -g cc-connect

# 登录各平台
cc-connect login feishu
cc-connect login dingtalk
cc-connect login slack

# 发送消息
cc-connect send --platform telegram --to CHAT_ID "Hello"
```
