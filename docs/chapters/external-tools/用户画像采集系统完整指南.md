## 2. 平台覆盖与采集方案

### 2.1 平台分类

```
🟢 即装即用（无需额外认证）：
  微信公众号 → Camoufox
  Twitter/X → xurl（OAuth2）
  YouTube/B站 → yt-dlp
  GitHub → gh
  知乎 → mcporter + Jina
  微博 → 公开API
  LinkedIn → mcporter + Jina
  V2EX → 公开API
  全网搜索 → Exa via mcporter

🟡 需要配置Cookie：
  小红书 → xhs-cli
  雪球 → 股票数据API

🔴 需要反检测+Cookie（强反爬）：
  美团/饿了么 → Camoufox（反检测）
  淘宝/京东/拼多多 → Camoufox（反检测）
  抖音 → Camoufox 或 mcporter MCP
```

### 2.2 各平台详细方案

#### Twitter/X

**工具**: xurl（已装，OAuth2认证）

```bash
# 认证
xfetch auth set --auth-token <token> --ct0 <token>

# 采集内容
xfetch user-tweets --username <user> --limit 100
xfetch search --query "AI machine learning" --limit 50

# 推文分析
xfetch engagement --username <user>  # 互动数据
xfetch followers --username <user>    # 粉丝画像
```

**采集数据类型**：推文内容、关注列表、粉丝、兴趣话题、互动频率

---

#### 微信公众号

**工具**: Camoufox（已装）

Camoufox 是反检测浏览器，能读取你已登录的 Chrome Cookie，绕过反爬机制。

```javascript
// 采集公众号文章
const { Camoufox } = require('camoufox');

const browser = await Camoufox.launch({
  headless: true,
  userDataDir: '~/.camoufox/profiles/default'
});

// 登录微信公众平台（读取Chrome Cookie）
await browser.loadCookies('chrome'); // 从Chrome读取Cookie

// 采集文章列表
const articles = await page.$$eval('.article_item', els => 
  els.map(el => ({
    title: el.querySelector('.title').innerText,
    date: el.querySelector('.date').innerText,
    url: el.querySelector('a').href
  }))
);
```

**采集数据类型**：文章标题、发布时间、阅读量、点赞数、评论

#### YouTube / Bilibili

**工具**: yt-dlp（已装）

```bash
# YouTube
yt-dlp --cookies-from-browser chrome \
  --output '%(title)s.%(ext)s' \
  --write-subs --write-auto-subs \
  https://www.youtube.com/feed/history

# B站
yt-dlp --cookies-from-browser chrome \
  --output '%(title)s.%(ext)s' \
  --write-subs \
  https://api.bilibili.com/x/web-interface/history/cursor

# 采集订阅列表
yt-dlp --flat-playlist \
  --dump-json \
  https://www.youtube.com/channel/<CHANNEL_ID>/videos
```

**采集数据类型**：观看历史、订阅列表、搜索记录、点赞视频、评论

---

#### GitHub

**工具**: gh CLI（已装）

```bash
# 认证
gh auth login

# 采集stars
gh api graphql -f query='
{
  viewer {
    starredRepositories(first: 100, orderBy: {field: STARRED_AT, direction: DESC}) {
      nodes {
        name
        description
        primaryLanguage { name }
        stargazerCount
        topics: repositoryTopics(first: 5) { nodes { topic { name } } }
      }
    }
  }
}' | jq '.data.viewer.starredRepositories.nodes[] | {name, lang: .primaryLanguage.name, stars: .stargazerCount, topics: [.topics.nodes[].topic.name]}'

# 采集贡献历史
gh api repos/<owner>/<repo>/contributors --jq '.[].login'

# 采集语言偏好
gh api graphql -f query='
{
  viewer {
    repositories(first: 100, orderBy: {field: UPDATED_AT, direction: DESC}) {
      nodes {
        name
        primaryLanguage { name }
        diskUsage
      }
    }
  }
}' | jq '[.data.viewer.repositories.nodes[] | .primaryLanguage.name] | group_by(.) | map({language: .[0], count: length}) | sort_by(-.count)'
```

**采集数据类型**：stars列表、repos语言分布、贡献活跃度、技术栈偏好

## 3. 工具配置详解

### 3.1 Camoufox 配置

Camoufox 是反检测浏览器，模拟真实用户行为，绕过反爬机制。

**安装**:
```bash
npm install -g camoufox
```

**基本配置**:
```javascript
const { Camoufox } = require('camoufox');

const browser = await Camoufox.launch({
  headless: false,
  userDataDir: '~/.camoufox/profiles/default',
  profile: {
    viewport: { width: 1920, height: 1080 },
    userAgent: 'Mozilla/5.0 (Macintosh; Intel Mac OS X 10_15_7)...',
    timezone: 'Asia/Shanghai',
    locale: 'zh-CN'
  },
  antibot: {
    enabled: true,
    undetectable: true
  }
});
```

**从Chrome读取Cookie**:
```javascript
// 自动从Chrome读取已登录Cookie
await browser.loadChromeCookies({
  profile: 'Default',
  cookies: [
    { domain: '.weixin.qq.com', name: 'wxuin' },
    { domain: '.xiaohongshu.com', name: 'web_session' }
  ]
});
```

### 3.2 yt-dlp 配置

**安装**:
```bash
brew install yt-dlp
```

**认证（读取Chrome Cookie）**:
```bash
yt-dlp --cookies-from-browser chrome \
  --auth-no-challenge \
  https://youtube.com
```

**高级用法**:
```bash
# B站登录态
yt-dlp --cookies-from-browser chrome \
  --write-subs --write-auto-subs \
  --sub-langs zh-Hans,en \
  --format "bv[ext=mp4]+ba[ext=m4a]" \
  https://www.bilibili.com/v/popular/history
```

### 3.3 xurl 配置

**认证（OAuth2）**:
```bash
xfetch auth set \
  --auth-token <your_auth_token> \
  --ct0 <your_ct0_token>
```

**验证**:
```bash
xfetch auth status
```

### 3.4 gh CLI 配置

**认证**:
```bash
gh auth login
# 选择 GitHub.com → HTTPS → Login with web browser
```

**配置**:
```bash
gh auth setup-git
export GITHUB_TOKEN=$(gh auth token)
```

## 5. 隐私与安全

### 5.1 Cookie 安全

| 注意 | 说明 |
|------|------|
| **不存储明文Cookie** | 使用环境变量或加密存储 |
| **定期刷新** | Cookie 有时效，定期更新 |
| **隔离环境** | 不同平台用不同 profile，避免关联 |

### 5.2 采集频率限制

| 平台 | 建议频率 | 原因 |
|------|---------|------|
| Twitter/X | ≤60次/小时 | API限制 |
| GitHub | ≤5000次/小时 | API限制 |
| 微信公众号 | 无明确限制 | 需反检测 |
| 美团/饿了么 | 低频 | 强反爬 |

### 5.3 数据保护

- 采集数据仅用于个人画像分析
- 不出售、不共享给第三方
- 定期清理过期数据

## 6. 常见问题

### Q1: Cookie 过期怎么办？
**A**: 重新在 Camoufox 浏览器中登录对应平台，导出新 Cookie。

### Q2: 账号被封禁？
**A**: 降低采集频率，使用多账号轮换。

### Q3: 反检测失败？
**A**: 更新 Camoufox 到最新版本，或使用 mcporter 作为备选。

### Q4: 数据不一致？
**A**: 各平台数据格式不同，需要标准化处理后再存储。
