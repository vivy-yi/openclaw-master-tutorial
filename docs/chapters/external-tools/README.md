## 为什么放在这里？

这些是 OpenClaw 生态之外的优秀工具，可以：

1. **参考设计**：FormKit 的 skill 自动配置思路可以借鉴到 OpenClaw
2. **集成学习**：了解如何构建开发者友好的工具生态
3. **技术拓展**：扩展技术视野，了解业界最佳实践
