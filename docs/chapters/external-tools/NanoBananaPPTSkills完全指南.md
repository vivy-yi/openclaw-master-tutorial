## 二、核心功能

### 2.1 功能总览

| 功能 | 说明 | 必需 |
|------|------|------|
| **智能文档分析** | 自动提取要点，规划结构 | ✅ |
| **PPT 图片生成** | Nano Banana Pro 生成 16:9 图片 | ✅ |
| **转场视频生成** | 可灵 AI 生成过渡动画 | ❌ |
| **交互式播放器** | HTML5 视频+图片混合播放 | ❌ |
| **完整视频导出** | FFmpeg 合成最终视频 | ❌ |

### 2.2 两种视觉风格

#### 渐变毛玻璃卡片风格
- **视觉**：Apple Keynote 极简主义、玻璃拟态、霓虹渐变
- **配色**：电光蓝 / 霓虹紫 / 珊瑚橙
- **适用**：科技产品、商务演示、数据报告

#### 矢量插画风格
- **视觉**：扁平化设计、黑色轮廓线、复古柔和配色
- **配色**：温暖色调、几何简化
- **适用**：教育培训、创意提案、品牌故事

## 三、工作流程

### 3.1 六阶段流程

```
┌─────────────────────────────────────────────────────────────────┐
│  阶段 1: 收集用户输入                                            │
│  • 文档路径 / 直接文本 / 主动询问                                │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│  阶段 2: 选择配置                                                │
│  • 风格：毛玻璃 / 矢量插画                                       │
│  • 页数：5 / 10 / 15 / 25 页                                    │
│  • 分辨率：2K / 4K                                              │
│  • 视频：是 / 否                                                │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│  阶段 3: 文档分析与内容规划                                      │
│  • 提取核心要点                                                 │
│  • 生成 slides_plan.json                                        │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│  阶段 4: 生成 PPT 图片                                           │
│  • Nano Banana Pro API                                          │
│  • ~30秒/页 (2K)                                               │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│  阶段 5: 生成转场视频（可选）                                    │
│  • 分析相邻图片差异                                             │
│  • 可灵 AI 生成过渡动画                                         │
│  • FFmpeg 合成完整视频                                          │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│  阶段 6: 返回结果                                                │
│  • 输出目录 + 播放网页                                          │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 内容规划策略

| 页数 | 时长 | 适用场景 |
|------|------|---------|
| **5 页** | 5 分钟 | 快速汇报 |
| **5-10 页** | 10-15 分钟 | 常规演示 |
| **10-15 页** | 20-30 分钟 | 深入讲解 |
| **20-25 页** | 45-60 分钟 | 完整培训 |

**5 页结构**：
1. 封面：标题 + 核心主题
2. 要点 1：第一个核心观点
3. 要点 2：第二个核心观点
4. 要点 3：第三个核心观点
5. 总结：核心结论或行动建议

## 四、安装与配置

### 4.1 安装方式

#### Claude Code 自动安装（推荐）
```
请帮我安装 NanoBanana PPT Skills：

1. 克隆项目并进入目录：
   git clone https://github.com/op7418/NanoBanana-PPT-Skills.git
   cd NanoBanana-PPT-Skills

2. 创建 Python 虚拟环境并安装依赖

3. 配置 API 密钥到 .env 文件

我的 API 密钥：
- GEMINI_API_KEY: YOUR_KEY
- KLING_ACCESS_KEY: YOUR_KEY (可选)
- KLING_SECRET_KEY: YOUR_KEY (可选)
```

#### 手动安装
```bash
git clone https://github.com/op7418/NanoBanana-PPT-Skills.git
cd NanoBanana-PPT-Skills

# 创建虚拟环境
python3 -m venv venv
source venv/bin/activate

# 安装依赖
pip install google-genai pillow python-dotenv

# 配置 API 密钥
cp .env.example .env
nano .env
```

### 4.2 API 密钥配置

```bash
# .env 文件
GEMINI_API_KEY=your_gemini_api_key_here    # 必需
KLING_ACCESS_KEY=your_kling_access_key      # 可选（视频）
KLING_SECRET_KEY=your_kling_secret_key      # 可选（视频）
```

**获取方式**：
| API | 获取地址 |
|-----|---------|
| Google AI | https://aistudio.google.com/apikey |
| 可灵 AI | https://klingai.com |

## 五、使用命令

### 5.1 生成 PPT 图片

```bash
# 基本用法
python generate_ppt.py \
  --plan slides_plan.json \
  --style styles/gradient-glass.md \
  --resolution 2K

# 或使用 uv run
uv run python generate_ppt.py \
  --plan slides_plan.json \
  --style styles/gradient-glass.md \
  --resolution 2K
```

### 5.2 生成转场视频

```bash
python generate_ppt_video.py \
  --slides-dir outputs/.../images \
  --output-dir outputs/..._video \
  --prompts-file outputs/.../transition_prompts.json
```

### 5.3 Skill 调用方式

```bash
/ppt-generator-pro
```

或直接描述需求：
```
我想基于以下文档生成一个 5 页的 PPT，使用渐变毛玻璃风格。

[文档内容...]
```

### 7.1 API 配置
