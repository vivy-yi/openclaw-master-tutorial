# 第11章 工作空间 (Workspace)

> **本章学习目标**: 理解 OpenClaw 工作空间的核心理念、类型划分和实践应用
> **预计用时**: 45-60分钟
> **前置要求**: 已完成第2章部署

---

## 本章概述

**工作空间 (Workspace)** 是 OpenClaw 的核心组织单元，它将所有垂直领域内容和服务全部集中在一起，保证隔离性的同时共享基础模块。

---

```

11.4 配置              → 学会配置多工作空间

11.6 最佳实践          → 目录组织 + 性能优化

```

---

## 核心概念

| 概念 | 说明 |
|------|------|
| **隔离性** | 不同工作空间之间完全独立，互不干扰 |
| **共享基础模块** | OpenClaw 内核、内置工具、认证系统等共用 |
| **组成要素** | Agent、Tools、Skills、Memory、Context、外部API、流程、规则 |
