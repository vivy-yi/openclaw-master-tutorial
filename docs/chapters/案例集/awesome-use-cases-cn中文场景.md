### 🤖 私人助理 / Life OS (5个)

这类案例致力于将 OpenClaw 打造成个人的 AI 私人助理，整合日程、待办、记忆等能力。

#### 1️⃣ 飞书私人助理

> 把 OpenClaw 变成日常协作中枢

| 项目 | 内容 |
|------|------|
| **一句话说明** | 把 OpenClaw 变成日常协作中枢，日程/待办/记忆一体化 |
| **适用渠道** | 飞书 (Feishu) |
| **核心功能** | 日程管理、待办同步、记忆存储、协作中枢 |
| **适用场景** | 个人工作流管理、日常团队协作 |

**典型工作流**:
1. 自然语言创建日程 → 自动同步到飞书日历
2. 待办事项一键同步到飞书任务
3. 重要信息用自然语言保存到记忆库
4. 通过对话查询历史记录和工作进展

## 🔧 快速开始

```bash
# 1. 安装需要的 Skill
npx clawhub@latest install feishu-doc
npx clawhub@latest install calendar-sync

# 2. 配置渠道
openclaw channels add <channel-name>

# 3. 首次对话
# 向你的 OpenClaw 发送: "帮我安排明天下午3点的会议"
```
