## 14.7.1 SSH密钥扫描器

### 场景描述

扫描代码库发现暴露的SSH密钥。

```bash
openclaw cron add \
  --name "SSH Key Scanner" \
  --cron "0 3 * * 0" \
  --message "扫描SSH密钥：
1. 扫描代码库
2. 检测私钥暴露
3. 报告风险位置
4. 建议修复步骤" \
  --announce
```
