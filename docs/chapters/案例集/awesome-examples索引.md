## 🚀 快速开始

```bash
# 1. 安装所需 Skill
npx clawhub@latest install <skill-slug>

# 2. 运行先决条件检查
./scripts/check_prereqs.sh

# 3. 查看示例输出
cat sample-output.md

# 4. 应用提示词并设置定时任务
cron "0 9 * * *" openclaw run <skill-name>
```
