## 5. 错误处理

### 5.1 错误类型

```
┌─────────────────────────────────────────────────────────────┐
│                   错误处理                                   │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  状态码        │ 错误类型              │ 说明        │   │
│  ├─────────────────────────────────────────────────────┤   │
│  │  400          │ BadRequestError        │ 请求错误    │   │
│  │  401          │ AuthenticationError   │ 认证失败    │   │
│  │  403          │ PermissionDeniedError │ 权限不足    │   │
│  │  404          │ NotFoundError         │ 未找到      │   │
│  │  422          │ UnprocessableEntity   │ 实体无法处理│   │
│  │  429          │ RateLimitError        │ 速率限制    │   │
│  │  >=500        │ InternalServerError   │ 服务器错误  │   │
│  │  N/A          │ APIConnectionError    │ 连接错误    │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                              │
│  处理示例:                                                  │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  const response = await client.add({               │   │
│  │    content: 'content'                              │   │
│  │  }).catch(async (err) => {                         │   │
│  │    if (err instanceof Supermemory.APIError) {     │   │
│  │      console.log(err.status);  // 400             │   │
│  │      console.log(err.name);     // BadRequestError│   │
│  │      console.log(err.headers);  // {...}          │   │
│  │    } else {                                        │   │
│  │      throw err;                                    │   │
│  │    }                                               │   │
│  │  });                                               │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```
