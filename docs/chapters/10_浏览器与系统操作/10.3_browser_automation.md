## 3.2 browser - 浏览器控制

### 功能说明

控制浏览器执行各种自动化操作。

### 参数

```json5
{
  "action": "click",           // 操作类型
  "selector": "#submit-btn",   // CSS 选择器
  "options": {
    "timeout": 30000,
    "visible": true
  }
}
```

### 操作类型

| 操作 | 说明 |
|------|------|
| navigate | 导航到 URL |
| click | 点击元素 |
| fill | 填写表单 |
| submit | 提交表单 |
| screenshot | 截图 |
| evaluate | 执行 JavaScript |
| wait | 等待元素 |

---

## 3.3 常用操作示例

### 导航操作

```bash
# 导航到 URL
browser_navigate("https://example.com")

# 带选项的导航
browser_navigate("https://example.com", {
  timeout: 30000,
  waitUntil: "networkidle"
})
```

### 点击操作

```bash
# 点击元素
browser_click("#submit-button")

# 点击链接
browser_click("a.login-link")

# 等待可见后点击
browser_click(".modal-close", { waitForVisible: true })
```

### 表单操作

```bash
# 填写输入框
browser_fill("#username", "admin")
browser_fill("#password", "secret123")

# 填写并提交
browser_fill("#search-input", "查询内容")
browser_click("#search-button")

# 选择下拉框
browser_select("#country", "CN")
```

### 截图操作

```bash
# 截取整个页面
browser_screenshot()

# 截取特定元素
browser_screenshot(".content-area")

# 截取并保存
browser_screenshot({ saveTo: "screenshots/login.png" })
```

### JavaScript 执行

```bash
# 执行 JavaScript
browser_evaluate("""
  document.querySelector('#result').textContent
""")

# 滚动页面
browser_evaluate("window.scrollTo(0, document.body.scrollHeight)")
```

## 3.4 工具配置

### 全局配置

```json5
{
  tools: {
    allow: ["group:web"]
  }
}
```

### 按 Agent 配置

```json5
{
  agents: {
    list: [
      {
        id: "scraper",
        tools: {
          allow: ["group:web"],
          deny: ["browser_evaluate"]  // 禁止执行 JS
        }
      }
    ]
  }
}
```

## 3.5 等待策略

### 等待选项

```json5
{
  "options": {
    "timeout": 30000,        // 超时时间
    "waitForVisible": true,   // 等待元素可见
    "waitForHidden": false,  // 等待元素隐藏
    "waitForNavigation": true // 等待导航完成
  }
}
```

### 等待策略类型

| 策略 | 说明 |
|------|------|
| load | 等待 load 事件 |
| domcontentloaded | 等待 DOMContentLoaded |
| networkidle | 等待网络空闲 |
| networkidle0 | 等待 0 个网络请求 |

## 3.6 错误处理

```bash
try {
  browser_navigate("https://example.com")
  browser_fill("#username", "admin")
  browser_fill("#password", "password123")
  browser_click("#login-btn")
} catch (error) {
  print("Browser error: " + error.message)
  browser_screenshot({ saveTo: "error.png" })
}
```

## 3.7 安全建议

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                       浏览器自动化安全建议                                      │
├─────────────────────────────────────────────────────────────────────────────────┤
│                                                                                 │
│  ✅ 推荐做法：                                                                  │
│  1. 限制可访问的域名                                                          │
│  2. 禁用危险操作（eval, localStorage）                                       │
│  3. 设置合理的超时                                                            │
│  4. 处理意外弹窗                                                              │
│                                                                                 │
│  ❌ 避免做法：                                                                  │
│  1. 在生产环境无限制访问互联网                                               │
│  2. 执行未验证的用户提供的脚本                                                │
│  3. 允许访问敏感域名                                                         │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘
```

### 域名限制配置

```json5
{
  agents: {
    defaults: {
      browser: {
        // 允许的域名
        allowedDomains: ["*.example.com"],
        // 禁止的域名
        blockedDomains: ["*.internal.com"]
      }
    }
  }
}
```
