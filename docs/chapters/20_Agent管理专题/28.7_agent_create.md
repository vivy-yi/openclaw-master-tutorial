# 7.7 Agent 创建与管理

## 本节目标
- 掌握创建单个 Agent 的方法
- 理解 Agent 的核心概念
- 学会配置 Agent 工作空间

---

## 7.7.1 Agent 核心概念

### 什么是 Agent

```
┌─────────────────────────────────────────────────────────────┐
│                       Agent 是什么                           │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  Agent = 独立的大脑，包含：                                 │
│  ├── Workspace (工作空间)                                   │
│  │   └── 文件、AGENTS.md/SOUL.md/USER.md                   │
│  │   └── 本地笔记、角色规则                                  │
│  ├── State Directory (状态目录)                              │
│  │   └── 认证配置、模型注册表                                │
│  └── Session Store (会话存储)                                │
│      └── 聊天历史、路由状态                                   │
│                                                              │
│  每个 Agent 完全隔离，互不影响                               │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### Agent 路径结构

```
~/.openclaw/
├── openclaw.json          # 全局配置
├── credentials/           # 认证凭据
├── skills/               # 共享 Skills
├── workspace/            # 默认工作空间 (main agent)
├── agents/
│   └── <agentId>/
│       ├── agent/        # Agent 状态目录
│       └── sessions/     # 会话存储
└── cron/                 # 定时任务
```

## 7.7.2 创建 Agent

### 使用 CLI 创建

```bash
# 创建新的 Agent
openclaw agents add work

# 创建多个 Agents
openclaw agents add coding
openclaw agents add social
openclaw agents add finance
```

### 查看 Agent 列表

```bash
# 列出所有 Agents
openclaw agents list

# 查看绑定信息
openclaw agents list --bindings
```

### 手动配置

在 `openclaw.json` 中配置：

```json
{
  "agents": {
    "list": [
      {
        "id": "main",
        "default": true,
        "name": "Main Agent",
        "workspace": "~/.openclaw/workspace"
      },
      {
        "id": "work",
        "name": "Work Agent",
        "workspace": "~/.openclaw/workspace-work"
      },
      {
        "id": "coding",
        "name": "Coding Agent",
        "workspace": "~/.openclaw/workspace-coding"
      }
    ]
  }
}
```

## 7.7.3 Agent 工作空间

### 工作空间文件结构

```
~/.openclaw/workspace/
├── AGENTS.md        # Agent 操作指令
├── SOUL.md          # 人设、语调
├── USER.md          # 用户信息
├── IDENTITY.md      # Agent 身份
├── TOOLS.md         # 工具说明
├── HEARTBEAT.md     # 心跳检查清单
├── BOOT.md          # 启动检查清单
├── memory/          # 每日记忆
│   ├── 2024-01-15.md
│   └── 2024-01-16.md
├── MEMORY.md        # 长期记忆
├── skills/          # 工作空间 Skills
└── canvas/         # Canvas UI 文件
```

### 每个文件的作用

| 文件 | 作用 | 加载时机 |
|-----|------|---------|
| AGENTS.md | Agent 行为规则 | 每次会话 |
| SOUL.md | 人设和边界 | 每次会话 |
| USER.md | 用户信息 | 每次会话 |
| IDENTITY.md | Agent 身份 | 启动时 |
| HEARTBEAT.md | 心跳检查清单 | 心跳运行时 |
| BOOT.md | 启动检查清单 | Gateway 重启时 |

### 配置工作空间

```json
{
  "agents": {
    "defaults": {
      "workspace": "~/.openclaw/workspace"
    }
  }
}
```

## 7.7.4 Agent 隔离配置

### 独立认证

每个 Agent 有独立的认证配置：

```
~/.openclaw/agents/<agentId>/agent/auth-profiles.json
```

### 共享认证

如需共享认证，手动复制：

```bash
# 复制认证配置
cp ~/.openclaw/agents/main/agent/auth-profiles.json \
   ~/.openclaw/agents/work/agent/auth-profiles.json
```

### 每个 Agent 的 Skills

- 工作空间 Skills：`~/.openclaw/workspace-<agentId>/skills/`
- 共享 Skills：`~/.openclaw/skills/`

## 7.7.5 Agent 工具限制

### 工具白名单/黑名单

```json
{
  "agents": {
    "list": [
      {
        "id": "family",
        "workspace": "~/.openclaw/workspace-family",
        "tools": {
          "allow": ["read", "sessions_list", "sessions_history"],
          "deny": ["exec", "write", "edit", "browser", "cron"]
        }
      }
    ]
  }
}
```

### 沙箱隔离

```json
{
  "agents": {
    "list": [
      {
        "id": "sandboxed",
        "workspace": "~/.openclaw/workspace-sandboxed",
        "sandbox": {
          "mode": "all",
          "scope": "agent"
        }
      }
    ]
  }
}
```

## 7.7.6 管理 Agent

### 常用命令

```bash
# 查看 Agent 状态
openclaw agents status

# 查看 Agent 会话
openclaw sessions list --agent work

# 删除 Agent
openclaw agents remove <agentId>

# 重置 Agent
openclaw agents reset <agentId>
```

---

## 本节小结

1. **Agent 概念**：独立的大脑，包含工作空间、状态目录、会话存储
2. **创建**：`openclaw agents add <name>`
3. **工作空间**：包含 AGENTS.md、SOUL.md、USER.md 等关键文件
4. **隔离**：每个 Agent 独立的认证和会话
5. **工具限制**：支持白名单/黑名单和沙箱隔离
