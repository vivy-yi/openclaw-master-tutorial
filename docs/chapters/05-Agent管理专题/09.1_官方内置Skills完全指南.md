## 四、使用方式

### 通过 `available_skills` 调用

```markdown
# 使用前先检查可用
<available_skills>
  <skill>
    <name>obsidian</name>
    <location>~/.openclaw/skills/obsidian/SKILL.md</location>
  </skill>
</available_skills>
```

### 通过 skill 名调用

```bash
# 在对话中触发
使用 skill: obsidian 同步笔记
使用 skill: weather 查询天气
```

## 五、自定义 Skill 存放位置

| 目录 | 作用域 |
|------|--------|
| `~/.openclaw/workspace/skills/` | 主 Agent 专用 |

| `~/.agents/skills/` | ClawHub 安装 |
