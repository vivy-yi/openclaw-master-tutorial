## 1.2 平台详细对比

| 平台 | 连接方式 | 消息类型 | 特色功能 | 推荐场景 |
|------|---------|---------|---------|---------|
| 飞书 | WebSocket | 文本/卡片/图片/文件 | 多维表格/知识库/任务 | 企业协作 |
| 钉钉 | Stream WS | 文本/卡片/图片/文件 | 互动卡片/引用消息 | 企业办公 |
| 企业微信 | WebSocket/回调 | 文本/图片/文件 | 主动发送/群聊 | 微信生态 |
| QQ | WebSocket | 文本/图片/文件 | 引用消息/主动发送 | 社群运营 |

## 1.4 各平台详细功能

### 飞书特色

- 多维表格集成 (feishu_bitable)
- 知识库文档读取 (feishu_wiki)
- 飞书机器人卡片消息
- 任务管理集成 (feishu_task)
- 审批流集成
- @ 消息转发
- 动态 Agent 创建

### 钉钉特色

- Stream 模式 WebSocket 长连接（无需公网 IP）
- AI 互动卡片（流式更新）
- 引用消息（文字/图片/文件/视频）
- 附件文本抽取（PDF/DOCX）
- 反馈学习系统
- 多账号支持
- Markdown 表格兼容

### 企业微信特色

- 智能机器人 WebSocket 长连接
- 自建应用支持主动发送
- 群聊和私聊支持
- 动态 Agent
- 工作区模板
- 运行时配额感知

### QQ 特色

- OneBot 11 协议
- NapCat 框架
- 引用消息上下文
- 已知目标管理
- 快速中断功能
- 语音转文字（腾讯云 ASR）

---

## 1.5 插件安装命令

```bash
# 飞书
openclaw plugins install @m1heng-clawd/feishu

# 钉钉
openclaw plugins install @soimy/dingtalk

# 企业微信
openclaw plugins install @sunnoy/wecom

# 统一安装（所有中国平台）
openclaw plugins install @openclaw-china/channels

# 配置向导
openclaw china setup
```

---

## 本节小结

1. **飞书** - 功能最全，企业协作首选
2. **钉钉** - 企业办公生态，AI 互动卡片
3. **企业微信** - 微信生态，支持主动发送
4. **QQ** - 社群运营，OneBot 协议
