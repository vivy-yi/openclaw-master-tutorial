# 5.6.3 浏览器操作

本节讲解 OpenClaw 浏览器的各种操作，包括导航、点击、输入、等待等。

---

## 5.6.3.1 基础操作命令

### 启动与停止

```bash
# 启动浏览器
openclaw browser start
openclaw browser --browser-profile openclaw start

# 停止浏览器
openclaw browser stop

# 查看状态
openclaw browser status

# 查看帮助
openclaw browser --help
```

---

## 5.6.3.2 Tab 管理

```bash
# 列出所有 Tab
openclaw browser tabs
openclaw browser tab

# 打开新 Tab
openclaw browser open https://example.com
openclaw browser tab new https://example.com

# 聚焦 Tab（通过序号或 targetId）
openclaw browser focus 2
openclaw browser focus abcd1234

# 关闭 Tab
openclaw browser close 2
openclaw browser close abcd1234
```

### Tab 选择快捷方式

```bash
# 选择第 2 个 Tab
openclaw browser tab select 2

# 关闭第 2 个 Tab
openclaw browser tab close 2
```

---

## 5.6.3.3 导航操作

```bash
# 导航到 URL（当前 Tab）
openclaw browser navigate https://example.com

# 后退
openclaw browser back

# 前进
openclaw browser forward

# 刷新
openclaw browser reload
```

---

## 5.6.3.4 元素交互

### 核心概念：Ref

在执行交互操作前，需要先获取页面快照，从快照中获取元素的 **ref**（引用）：

```bash
# 获取页面快照
openclaw browser snapshot

# 输出示例:
# [12] 🔍 Search input
# [13] 🔗 Search button
# [14] 🔗 Result link 1
# [15] 🔗 Result link 2
```

### 点击操作

```bash
# 点击元素（通过 ref）
openclaw browser click 12

# 双击
openclaw browser click 12 --double

# 右键点击
openclaw browser click 12 --button right
```

### 输入操作

```bash
# 输入文本
openclaw browser type 12 "Hello World"

# 输入并提交（自动按 Enter）
openclaw browser type 12 "Hello World" --submit
```

### 按键操作

```bash
# 按键
openclaw browser press Enter
openclaw browser press Tab
openclaw browser press Escape
openclaw browser press "Control+c"

# 组合键
openclaw browser press "Control+Shift+t"
```

### 悬停操作

```bash
# 悬停到元素
openclaw browser hover 12
```

### 拖拽操作

```bash
# 拖拽（从 ref 1 拖到 ref 2）
openclaw browser drag 10 11
```

### 选择操作

```bash
# 下拉框选择
openclaw browser select 9 OptionA OptionB
```

## 5.6.3.5 表单填写

### JSON 方式批量填充

```bash
openclaw browser fill --fields '[
  {"ref": "1", "type": "text", "value": "Ada"},
  {"ref": "2", "type": "text", "value": "Lovelace"},
  {"ref": "3", "type": "email", "value": "ada@example.com"}
]'
```

### 字段类型

| 类型 | 说明 |
|------|------|
| `text` | 文本输入 |
| `email` | 邮箱输入 |
| `password` | 密码输入 |
| `checkbox` | 复选框 |
| `radio` | 单选框 |
| `select` | 下拉框 |

## 5.6.3.6 等待操作

等待是浏览器自动化的关键，OpenClaw 提供强大的等待机制。

### 等待时间

```bash
# 等待指定毫秒
openclaw browser wait --time 3000
```

### 等待文本

```bash
# 等待文本出现
openclaw browser wait --text "Search Results"
```

### 等待选择器

```bash
# 等待选择器元素出现
openclaw browser wait "#results"
openclaw browser wait ".loading-spinner"

# 等待元素可见
openclaw browser wait "#main" --state visible
```

### 等待 URL

```bash
# 等待 URL 匹配（支持 glob）
openclaw browser wait --url "**/dash"
openclaw browser wait --url "*example.com*"
```

### 等待加载状态

```bash
# 等待网络空闲
openclaw browser wait --load networkidle

# 等待 DOM 加载完成
openclaw browser wait --load domcontentloaded

# 等待完整加载
openclaw browser wait --load load
```

### 等待 JavaScript 条件

```bash
# 等待 JS 表达式返回 true
openclaw browser wait --fn "window.ready===true"
openclaw browser wait --fn "document.querySelector('.loaded') !== null"
```

### 组合等待

```bash
# 组合多个条件
openclaw browser wait "#main" \
  --url "**/dash" \
  --load networkidle \
  --fn "window.ready===true" \
  --timeout-ms 15000
```

## 5.6.3.7 滚动操作

```bash
# 滚动到指定位置
openclaw browser scroll 0 500

# 执行 JavaScript 滚动
openclaw browser evaluate --fn "window.scrollTo(0, 500)"

# 滚动到元素
openclaw browser scrollintoview 12

# 滚动到底部
openclaw browser evaluate --fn "window.scrollTo(0, document.body.scrollHeight)"
```

---

## 5.6.3.8 JavaScript 执行

```bash
# 执行 JavaScript
openclaw browser evaluate --fn "document.title"

# 获取元素属性
openclaw browser evaluate --fn "document.querySelector('img').src"

# 点击元素（通过 JS）
openclaw browser evaluate --fn "document.querySelector('.btn').click()"

# 获取页面数据
openclaw browser evaluate --fn "JSON.stringify({title: document.title, url: location.href})"
```

## 5.6.3.14 完整示例：自动化搜索流程

```bash
# 1. 启动浏览器并打开搜索页面
openclaw browser start
openclaw browser open https://www.google.com

# 2. 获取页面快照，找到搜索框和按钮
openclaw browser snapshot
# 输出: [12] 🔍 Search input, [13] 🔗 Search button

# 3. 在搜索框输入文本
openclaw browser type 12 "OpenClaw AI"

# 4. 点击搜索按钮（或直接提交）
openclaw browser click 13
# 或者
openclaw browser type 12 "OpenClaw AI" --submit

# 5. 等待结果加载
openclaw browser wait --text "Results"

# 6. 截图保存
openclaw browser screenshot
```
