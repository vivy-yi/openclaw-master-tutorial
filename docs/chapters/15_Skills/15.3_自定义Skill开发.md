# 自定义 Skill 开发

> 如何创建自己的 Skill。

---

## 一、Skill 结构

```
my-skill/
├── SKILL.md          # 必需：Skill 定义文件
├── scripts/          # 可选：执行脚本
├── prompts/          # 可选：Prompt 模板
└── assets/           # 可选：资源文件
```

---

## 二、SKILL.md 结构

```markdown
---
name: my-skill
description: 我的自定义 Skill
emoji: 🚀
---

# My Skill

> 描述这个 Skill 做什么。

## 使用方式

1. 触发方式
2. 参数说明
3. 示例

## 工作原理

详细说明执行流程。
```

---

## 三、创建示例

### Step 1: 创建目录

```bash
mkdir -p ~/.openclaw/workspace/skills/my-skill
```

### Step 2: 编写 SKILL.md

```markdown
---
name: my-greeting
description: 友好问候技能
emoji: 👋
---

# My Greeting Skill

向用户发送友好问候。

## 触发方式

- "你好"
- "hello"
- "say hi"

## 示例

用户：你好
→ 回复：你好！有什么可以帮你的？
```

### Step 3: 验证

```bash
# 检查语法
cat ~/.openclaw/workspace/skills/my-greeting/SKILL.md
```

---

## 四、自定义 Skill 存放位置

| 位置 | 作用域 | 用途 |
|------|--------|------|
| `~/.openclaw/workspace/skills/` | 主 Agent 专用 | 个人开发 |
| `~/.openclaw/skills/` | 全局共享 | 团队共享 |
| `~/.agents/skills/` | ClawHub 安装 | 外部 Skills |

---

## 五、ClawHub 发布

```bash
# 打包发布
clawhub publish my-skill

# 设为公开
clawhub publish my-skill --public
```
