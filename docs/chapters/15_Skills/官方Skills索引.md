## 📖 使用指南

### 安装 Skills

```bash
# 通过 ClawHub 安装
npx clawhub@latest install <skill-name>

# 或者直接在配置文件中启用
openclaw skills enable <skill-name>
```

### 查看可用 Skills

```bash
# 列出所有已安装的 Skills
openclaw skills list

# 搜索 ClawHub
npx clawhub@latest search <keyword>
```

### 配置 Skills

大部分 Skills 需要在 `~/.openclaw/openclaw.json` 中配置相关参数（如 API Key、连接凭证等）。
