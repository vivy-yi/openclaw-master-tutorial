## 四、Skills vs Plugins vs Tools 对比

| 层级 | 说明 | 位置 | 复杂度 |
|------|------|------|--------|
| **Prompts** | 用户指令系统 | `docs/prompts/` | 低 |
| **Tools** | 工具文档 | `docs/tools/` | 低 |
| **Skills** | 轻量级任务封装 | `openclaw/skills/` | 中 |
| **Plugins** | 运行时扩展 | `openclaw/plugins/` | 高 |

## 五、使用方式

### 触发 Skills

```markdown
# 在对话中直接触发
使用 skill: obsidian 同步笔记到知识库
使用 skill: weather 查询上海天气

# 通过 available_skills 检查
<available_skills>
  <skill>
    <name>weather</name>
    <location>~/.openclaw/skills/weather/SKILL.md</location>
  </skill>
</available_skills>
```
