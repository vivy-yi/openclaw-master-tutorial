## 六、使用方式

### 触发 Skills

```markdown
# 在对话中直接触发
使用 skill: obsidian 同步笔记到知识库
使用 skill: weather 查询上海天气
使用 skill: law-contract 审查合同条款
使用 skill: stock-analysis 分析科技股

# 通过 available_skills 检查
<available_skills>
  <skill>
    <name>weather</name>
    <location>~/.openclaw/skills/weather/SKILL.md</location>
  </skill>
</available_skills>
```

### 命令行操作

```bash
# 检查 Skills 状态
openclaw skills check

# 搜索 Skills
openclaw skills search <关键词>

# 安装 ClawHub Skills
openclaw skills install <skill-name>

# 更新 Skills
openclaw skills update
```
