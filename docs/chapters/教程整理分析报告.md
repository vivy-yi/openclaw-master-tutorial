## 五、执行注意事项

1. **编号变更需要同步更新**：目录更名后，所有内部文件的相对引用（如 `[第X章](../xx_xxx/)`）不会自动更新，需要用脚本批量处理。
2. **LEARNING_PATH.md** 中引用的是相对路径（如 `chapters/03_发出指令/`），需要同步更新。
3. **如果有其他文档引用了这些章节目录**，变更前需确认影响范围。
4. **建议分步执行**：先做 P0 的目录重命名，再处理 P1，最后 P2/P3。
5. **重命名前备份**：或使用 git 管理，确保可回滚。
