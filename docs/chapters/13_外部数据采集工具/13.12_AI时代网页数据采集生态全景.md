## Layer 1: 基础内容读取

> 最底层的 HTTP 请求工具，将网页内容拉取到本地

### 工具对比

| 工具 | 类型 | 输出 | JavaScript | 难度 | 费用 |
|------|------|------|-----------|------|------|
| **curl** | CLI | 原始HTML | ❌ | ⭐ | 免费 |
| **wget** | CLI | 原始HTML | ❌ | ⭐ | 免费 |
| **requests** | Python库 | 原始HTML | ❌ | ⭐⭐ | 免费 |
| **aiohttp** | Python库 | 原始HTML | ❌ | ⭐⭐⭐ | 免费 |
| **httpx** | Python库 | 原始HTML | 可通过playwright | ⭐⭐ | 免费 |
| **Jina Reader** | API | Markdown | ✅ | ⭐⭐ | 免费 |

### Jina Reader 详解

```bash
# 最简单的方式 - curl 调用 Jina Reader
curl -s "https://r.jina.ai/https://example.com"

# 指定输出格式
curl -s "https://r.jina.ai/html/https://example.com"  # HTML
curl -s "https://r.jina.ai/text/https://example.com"   # 纯文本
```

**优点**：
- 输出 Markdown，直接用于 RAG
- 自动清理广告/噪声
- 免费无限制

**缺点**：
- 不支持动态内容
- 不支持登录态

### 适用场景

| 场景 | 推荐工具 |
|------|----------|
| 简单静态页面 | curl |
| 需要 Markdown 输出 | Jina Reader |
| 批量下载文件 | wget |

## Layer 2: AI 友好爬虫

> 在 Layer 1 基础上，增加 AI 友好的结构化输出

### 工具对比

| 工具 | 类型 | 输出 | 渲染JS | 登录 | 费用 |
|------|------|------|--------|------|------|
| **Firecrawl** | SaaS | Markdown/HTML | ✅ | ✅ | 付费 |
| **Crawl4AI** | 开源自托管 | Markdown | ✅ | ✅ | 免费 |
| **Jina AI** | API | 结构化 | ✅ | ❌ | 免费/付费 |
| **Trafilatura** | 开源 | 结构化 | ❌ | ❌ | 免费 |
| **Newspaper** | 开源 | 纯文本 | ❌ | ❌ | 免费 |

### Firecrawl

```python
from firecrawl import Firecrawl

fc = Firecrawl(api_key="...")
result = fc.crawl_url("https://example.com", formats=["markdown"])
print(result.markdown)
```

**优点**：
- SaaS 托管，无需维护
- 支持 JavaScript 渲染
- 支持登录态
- 输出结构化

**缺点**：
- 付费服务
- 依赖第三方

### Crawl4AI

```bash
# 自托管
docker run -d -p 8000:8000 --ghcr.io/ffdeparcrawl4ai:latest

# API 调用
curl -X POST "http://localhost:8000/crawl" \
  -d {"urls": ["https://example.com"]}
```

**优点**：
- 开源免费
- 可自托管
- 输出 Markdown

**缺点**：
- 需要自己维护

### Jina AI (Search + Crawl)

```python
from jina import Client

client = Client(token="...")
result = client.search("what is machine learning", limit=10)
```

**优点**：
- 搜索 + 爬取一体化
- 免费额度

**缺点**：
- 搜索是主要功能

### 适用场景

| 场景 | 推���工具 |
|------|----------|
| 需要托管服务 | Firecrawl |
| 需要自托管 | Crawl4AI |
| RAG 数据准备 | Jina Reader / Trafilatura |

## Layer 4: AI 自主控制

> AI Agent 自主决策控制浏览器

### 工具对比

| 工具 | 类型 | 自主程度 | 集成 | 费用 |
|------|------|----------|------|------|
| **browser-use** | 开源 | 高 | LangChain/LlamaIndex | 免费 |
| **OpenCLI** | 开源 | 中 | 多平台 | 免费 |
| **Agent Use** | SaaS | 高 | API | 付费 |

### browser-use

```python
from browser_use import Agent
from langchain_openai import ChatOpenAI

agent = Agent(
    task="Find the price of iPhone 15 on amazon.com",
    llm=ChatOpenAI(model="gpt-4"),
)

result = agent.run()
```

**工作原理**：
1. AI 分析页面
2. 自主决定下一步动作
3. 执行并评估结果
4. 重复直到完成

**优点**：
- 完全自主
- 支持复杂任务
- 开源免费

**缺点**：
- 需要配置 LLM
- 可能有 token 消耗

### OpenCLI

```bash
# 安装
pip install opencli

# 使用
opencli browse "Find the price of iPhone 15"
```

**优点**：
- 开箱即用
- 90+ 适配器
- 多模型支持

### 适用场景

| 场景 | 推荐工具 |
|------|----------|
| 复杂任务自动化 | browser-use |
| 快速原型 | OpenCLI |
| 商业服务 | Agent Use |

## Layer 5: 反检测增强

> 绕过网站反爬虫检测

### 工具对比

| 工具 | 类型 | 原理 | 效果 |
|------|------|------|------|
| **Camoufox** | C++ 分支 | 完全重写 Firefox | ⭐⭐⭐⭐⭐ |
| **Playwright Stealth** | 插件 | 伪造特征 | ⭐⭐⭐⭐ |
| **Puppeteer Extra** | 插件 | 插件增强 | ⭐⭐⭐ |
| **Undetected ChromeDriver** | 补丁 | 补丁修改 | ⭐⭐⭐ |

### Camoufox

```bash
# 安装
git clone https://github.com/ceaeck/camoufox
cd camoufox
cargo build --release

# 使用
./camoufox https://example.com
```

**特点**：
- 完全重写 Firefox 底层
- 模拟真实用户行为
- 反自动化检测最有效
- C++ 性能好

### Playwright Stealth

```python
from playwright_stealth import stealth
from playwright.sync_api import sync_playwright

with sync_playwright() as p:
    browser = p.chromium.launch()
    page = browser.new_page()
    stealth(page)  # 应用反检测
    page.goto("https://example.com")
```

### 反检测技术

| 技术 | 说明 |
|------|------|
| User-Agent 随机化 | 每次请求随机 UA |
| WebGL 指纹 | 伪造显卡信息 |
| Canvas 指纹 | 伪造 Canvas 渲染 |
| 行为模拟 | 模拟人类鼠标移动 |
| 时间间隔 | 随机延迟 |

### 适用场景

| 场景 | 推荐工具 |
|------|----------|
| 高防护网站 | Camoufox |
| 普通防护 | Playwright Stealth |
| 快速测试 | Undetected ChromeDriver |

## 选择指南

### 场景 → 工具映射

| 你的需求 | 推荐方案 |
|---------|----------|
| **简单网页转 Markdown** | curl + Jina Reader |
| **批量爬取静态页面** | Crawl4AI / Trafilatura |
| **需要 JavaScript 渲染** | Playwright |
| **复杂自动化任务** | browser-use |
| **反检测要求高** | Camoufox |
| **一站式解决方案** | OpenCLI / Agent Reach |
| **OpenClaw 用户** | OpenCLI CDP |

### 复杂度 → 方案

| 复杂度 | 方案 |
|--------|------|
| **★☆☆☆☆** | curl + Jina Reader |
| **★★☆☆☆** | Playwright |
| **★★★☆☆** | Playwright Stealth |
| **★★★★☆** | browser-use |
| **★★★★★** | Camoufox + browser-use |

### OpenClaw 用户专属

| 需求 | OpenClaw 工具 |
|------|--------------|
| 基础搜索 | web_search (Brave) |

## 技术栈组合示例

### 最小组合 (免费)

```bash
# 1. 基础抓取
curl "https://r.jina.ai/https://example.com"

# 2. 浏览器自动化
pip install playwright
playwright install

# 3. 反检测
pip install playwright-stealth
```

### 专业组合

