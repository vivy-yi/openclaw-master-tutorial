## 11.0 生态概览 & 分类地图

### 工具分类

外部数据采集工具按**数据获取方式**分为 4 类：

```
┌─────────────────────────────────────────────────────────────────┐
│              外部数据采集工具生态                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  🔒 反检测浏览器        → Camoufox（C++ Firefox）                │
│     模拟真实用户浏览器，指纹不可检测，绕过 WAF/反爬               │
│                                                                  │
│  📄 AI 友好爬虫         → Firecrawl / Crawl4AI / Jina           │
│     将网页转换为 LLM 可直接消费的格式（Markdown/JSON）           │
│                                                                  │
│  🤖 AI Agent 浏览器控制  → browser-use                           │
│     LLM 自主决策操作浏览器完成复杂任务                            │
│                                                                  │
│  🔌 MCP/CLI 适配器     → Playwright MCP / BB-Browser /          │
│                           Agent Reach / OpenCLI                  │
│     通过 MCP 协议或 CLI 将平台能力暴露给 AI Agent                │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 核心判断题

> **Q：我的数据需要登录态吗？**
> - ✅ 是 → Camoufox / BB-Browser / OpenCLI
> - ❌ 否 → Jina Reader / Firecrawl / Crawl4AI

> **Q：我的数据需要 AI 理解/结构化提取吗？**
> - ✅ 是 → Firecrawl / Crawl4AI / browser-use
> - ❌ 否 → Camoufox / OpenCLI / Jina Reader

> **Q：我的数据规模大吗？**
> - 大规模 → Crawl4AI（自托管免费）/ Firecrawl（云端弹性）
> - 小规模 → Jina Reader / OpenCLI
