# DDGS (Dux Distributed Global Search) 完全指南

> 本教程基于源码分析编写：https://github.com/deedy5/ddgs

## 什么是 DDGS？

DDGS 是一个**元搜索库 (metasearch library)**，它聚合了多种 web 搜索服务的结果。

与 Exa 的神经语义搜索不同，DDGS 是一个聚合搜索引警，支持多个搜索引擎：

| 引擎 | 说明 |
|------|------|
| Bing | Microsoft 搜索 |
| Google | Google 搜索 |
| DuckDuckGo | 隐私搜索 |
| Brave | Brave 搜索 |
| Yandex | 俄罗斯搜索 |
| Yahoo | Yahoo 搜索 |
| Wikipedia | 维基百科搜索 |

---

## 与 Exa 的对比

| 特性 | DDGS | Exa |
|------|------|-----|
| **搜索类型** | 元搜索（聚合多引擎） | 神经语义搜索 |
| **语义理解** | ❌ 关键词匹配 | ✅ 语义理解 |
| **代码搜索** | ❌ | ✅ |
| **DHT 网络** | ✅ P2P 分布式缓存 | ❌ |
| **免费额度** | 无限制（聚合各引擎） | 付费 |

**选择建议**：
- 需要代码/API 文档 → 用 Exa
- 需要快速网页搜索 → 用 DDGS
- 需要组合使用 → 参考「工具选择策略」章节

## 快速开始

### 1. Python API

```python
from ddgs import DDGS

# 文本搜索
results = DDGS().text("python programming", max_results=5)
print(results)
```

### 2. CLI

```bash
# 文本搜索
ddgs "python programming"

# 图片搜索
ddgs images "sunset"

# 提取网页内容
ddgs extract -u https://example.com
```

### 3. API 服务器

```bash
# 启动服务器（默认端口 4479）
ddgs api

# 后台运行
ddgs api -d

# 自定义端口
ddgs api --host 127.0.0.1 --port 4479

# 使用代理
ddgs api -pr socks5h://127.0.0.1:9150
```

**API 端点**：

| 端点 | 方法 | 说明 |
|------|------|------|
| `/search/text` | GET, POST | 文本搜索 |
| `/search/images` | GET, POST | 图片搜索 |
| `/search/news` | GET, POST | 新闻搜索 |
| `/search/videos` | GET, POST | 视频搜索 |
| `/search/books` | GET, POST | 图书搜索 |
| `/extract` | GET, POST | 提取网页内容 |
| `/health` | GET | 健康检查 |

## 搜索方法详解

### 1. text() - 文本搜索

```python
results = DDGS().text(
    query="python programming",
    region="us-en",           # 地区代码
    safesearch="moderate",   # on, moderate, off
    timelimit="y",           # d, w, m, y
    max_results=10,
    page=1,
    backend="auto"          # auto, bing, google, duckduckgo, etc.
)
```

**返回格式**：
```python
[
    {
        "title": "标题",
        "href": "https://example.com",
        "body": "摘要内容..."
    },
    ...
]
```

### 2. images() - 图片搜索

```python
results = DDGS().images(
    query="butterfly",
    region="us-en",
    safesearch="off",
    timelimit="m",
    max_results=10,
    page=1,
    backend="auto",
    size="Large",           # Small, Medium, Large, Wallpaper
    color="Monochrome",    # color, Monochrome, Red, Orange, Yellow, Green, Blue, Purple, Pink, Brown, Black, Gray, Teal, White
    type_image="photo",     # photo, clipart, gif, transparent, line
    layout="Wide",         # Square, Tall, Wide
    license_image="any"    # any, Public, Share, ShareCommercially, Modify, ModifyCommercially
)
```

**返回格式**：
```python
[
    {
        "title": "标题",
        "image": "https://...",
        "thumbnail": "https://...",
        "url": "https://...",
        "height": 3860,
        "width": 4044,
        "source": "Bing"
    },
    ...
]
```

### 3. videos() - 视频搜索

```python
results = DDGS().videos(
    query="cars",
    region="us-en",
    safesearch="off",
    timelimit="w",
    max_results=10,
    page=1,
    backend="auto",
    resolution="high",     # high, standart
    duration="medium",     # short, medium, long
    license_videos="creativeCommon"  # creativeCommon, youtube
)
```

### 4. news() - 新闻搜索

```python
results = DDGS().news(
    query="technology",
    region="us-en",
    safesearch="off",
    timelimit="d",         # d, w, m
    max_results=10,
    page=1,
    backend="auto"
)
```

**返回格式**：
```python
[
    {
        "date": "2024-07-03T16:25:22+00:00",
        "title": "标题",
        "body": "摘要...",
        "url": "https://...",
        "image": "https://...",
        "source": "Bloomberg on MSN.com"
    },
    ...
]
```

### 5. books() - 图书搜索

```python
results = DDGS().books(
    query="sea wolf jack london",
    max_results=10,
    page=1,
    backend="auto"
)
```

**返回格式**：
```python
[
    {
        "title": "The Sea-Wolf",
        "author": "Jack London",
        "publisher": "DigiCat, 2022",
        "info": "English [en], .epub, 🚀/zlib, 0.5MB, 📗 Book (unknown)",
        "url": "https://annas-archive.li/...",
        "thumbnail": "https://..."
    },
    ...
]
```

### 6. extract() - 提取网页内容

```python
# 提取网页内容（默认 Markdown 格式）
result = DDGS().extract("https://example.com")
print(result)
# {"url": "https://example.com", "content": "# Example Domain\n\n..."}

# 其他格式
result = DDGS().extract("https://example.com", fmt="text_plain")   # 纯文本
result = DDGS().extract("https://example.com", fmt="text_rich") # 富文本
result = DDGS().extract("https://example.com", fmt="text")     # 原始 HTML
result = DDGS().extract("https://example.com", fmt="content")  # 原始字节
```

**format 参数**：
| 值 | 说明 |
|-----|------|
| `text_markdown` | HTML → Markdown（默认）|
| `text_plain` | HTML → 纯文本 |
| `text_rich` | HTML → 富文本 |
| `text` | 原始 HTML |
| `content` | 原始字节 |

## DHT 网络（测试版）

DDGS 独有的 **P2P 分布式缓存网络**，可以让搜索结果在用户之间共享。

### 特性

- ✅ **90% faster** 重复查询（50ms vs 1-2s）
- ✅ **无 rate limit** 公共查询
- ✅ **零基础设施成本** - 运行在用户机器上
- ✅ **匿名** - 不记录日志

### 安装

```bash
pip install -U ddgs[dht]

# Linux/macOS 额外依赖
pip install coincurve@git+https://github.com/ofek/coincurve.git@7829b29c08ebb1cc80386a1cdaf8c2243c4ef5c5
pip install libp2p@git+https://github.com/libp2p/py-libp2p.git@0e88584c89377086883c6f5b26cd1a8052399be7

# macOS 需要先安装 gmp
brew install gmp

# Windows 不支持 DHT
```

### 使用

```python
from ddgs import DDGS

# 初始化 DHT（自动启动）
ddgs = DDGS(api_url="http://localhost:4479", spawn_api=True)

# 现有代码无需修改，自动使用 DHT 缓存
results = DDGS().text("python")
```

### DHT API 端点

| 端点 | 说明 |
|------|------|
| `/dht/cache` | 获取/存储/删除缓存 |
| `/dht/status` | DHT 状态和指标 |
| `/dht/peers` | 连接的节点 |
| `/dht/peers/detailed` | 详细节点信息 |
| `/dht/map` | 网络图 |
| `/dht/metrics` | Prometheus 指标 |

## 常见问题

### Q1: 搜索结果为空
- 检查网络连接
- 尝试不同的 backend
- 增加 max_results

### Q2: Rate limit
- 使用 DHT 网络缓存
- 换用其他搜索引擎
- 使用代理

### Q3: DHT 无法启动
- 确认 Linux/macOS
- 检查依赖安装
- 查看日志
