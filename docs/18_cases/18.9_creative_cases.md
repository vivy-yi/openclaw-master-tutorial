## 14.9.1 目标驱动自主任务

### 场景描述

睡前倾倒目标，醒来发现AI已经自动完成。

### 配置

```
提示词：
作为自主任务助手，我睡觉时请：
1. 分析我列出的目标
2. 分解为可执行的任务
3. 按优先级排序
4. 自动执行不需要我确认的部分
5. 需要确认的部分标记

执行完成后报告进展
```

## 14.9.2 YouTube内容管道

### 场景描述

自动化视频创意研究和内容跟踪。

### 配置

```bash
openclaw cron add \
  --name "YouTube Pipeline" \
  --cron "0 */4 * * *" \
  --message "YouTube内容管道：
1. 扫描竞争对手频道
2. 提取热门视频主题
3. 生成创意建议
4. 更新内容日历" \
  --no-deliver
```
