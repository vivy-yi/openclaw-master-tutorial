## 14.12.1 Polymarket自动驾驶

### 功能

- 自动交易
- 回测分析
- 策略分析
- 每日报告

### 配置

```bash
openclaw cron add \
  --name "Polymarket Autopilot" \
  --cron "0 */4 * * *" \
  --message "交易策略执行：
1. 扫描市场机会
2. 执行交易
3. 风险管理检查
4. 生成日报" \
  --announce
```
