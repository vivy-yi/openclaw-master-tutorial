## 14.3.1 医疗邮件转播客

### 场景描述

将医疗健康类Newsletter转换为音频，方便通勤收听。

### 配置

```bash
openclaw cron add \
  --name "Medical Email to Podcast" \
  --cron "0 7 * * 1" \
  --message "将本周医疗Newsletter转换为播客：
1. 提取关键文章
2. 总结核心要点
3. 生成语音文件
4. 发送到Telegram" \
  --announce
```

## 14.3.2 邮件转播客技能

### 场景描述

通用的邮件转语音技能，可重复使用。

### 提示词模板

```
将以下邮件内容转换为播客格式：
1. 提取每个 Newsletter 的标题和摘要
2. 挑选 3-5 篇最值得关注的文章
3. 每篇生成 2-3 分钟的语音内容
4. 使用友好的播客风格

输出：MP3 文件或音频链接
```
