## 14.2.1 每日早间简报 (最受欢迎)

### 场景描述

每天早上醒来收到一条个性化简报：天气+日历+新闻+待办。

### 完整配置

```bash
openclaw cron add \
  --name "Morning Brief" \
  --cron "30 6 * * *" \
  --tz "Asia/Shanghai" \
  --session isolated \
  --message "你是我的早间简报助手。请执行：

1. 天气：获取今日北京天气
2. 日历：列出今日日程
3. 新闻：3条科技头条
4. 待办：今日待办事项

格式：☀️ 天气 | 📅 日历 | 📰 新闻 | ✅ 待办
300字内" \
  --announce \
  --channel telegram
```

### 效果示例

```
☀️ 北京 | 晴 22-28°C 无需带伞
📅 10:00团队周会 | 14:00客户演示
📰 1.OpenAI发布新模型 2.特斯拉Q3财报
✅ 发合同 | 回邮件 | 准备PPT
```

## 14.2.2 邮件自动分类

### 场景描述

按紧急程度自动分类收件箱，优先处理重要邮件。

### 配置

```bash
openclaw cron add \
  --name "Email Sort" \
  --cron "0 * * * *" \
  --message "分类收件箱邮件：
1. 紧急：需要立即处理
2. 重要：今天需要回复
3. 普通：本周处理
4. 归档：无需处理

使用Gmail CLI执行" \
  --no-deliver
```

## 14.2.3 智能日历提醒

### 场景描述

会议前自动提醒，并提供准备建议。

### 提示词

```
会议前30分钟提醒我：
1. 会议主题和参与者
2. 相关的历史讨论要点
3. 我需要准备的资料
4. 可能的讨论议题

根据会议类型提供：
- 1v1：准备进度更新
- 团队会议：准备议程
- 客户演示：准备Demo
```

## 14.2.5 习惯追踪教练

  --announce
