## 2.9.2 Hetzner 部署

### 为什么选择 Hetzner

```
┌─────────────────────────────────────────────────────────────┐
│                    Hetzner 优势                              │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  💰 价格实惠                                                │
│     └── Cloud€4/月起，性价比最高                            │
│                                                              │
│  🇪🇺 欧洲节点                                               │
│     └── 延迟低，适合欧洲用户                                 │
│                                                              │
│  🚀 性能好                                                  │
│     └── NVMe SSD，速度快                                    │
│                                                              │
│  🔧 API 友好                                                │
│     └── hcloud CLI，操作简单                                 │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 步骤一：注册账号

1. 访问 [Hetzner Cloud](https://hetzner.cloud/)
2. 注册账号（需要信用卡/PayPal）
3. 创建项目

### 步骤二：安装 hcloud CLI

```bash
# macOS
brew install hcloud

# Linux
curl -s https://get.hetzner.com | bash

# 登录
hcloud context create my-project
```

### 步骤三：创建服务器

```bash
# 创建服务器 (CPX11: 2CPU, 2GB RAM, 40GB NVMe)
hcloud server create --name openclaw \
  --image ubuntu-22.04 \
  --location nbg1 \
  --type cpx11 \
  --ssh-key ~/.ssh/id_rsa.pub
```

### 步骤四：配置防火墙

```bash
# 创建防火墙
hcloud firewall create --name openclaw -n openclaw

# 开放必要端口
hcloud firewall rule add openclaw --protocol tcp --port 22   # SSH
hcloud firewall rule add openclaw --protocol tcp --port 18789 # OpenClaw
hcloud firewall rule add openclaw --protocol tcp --port 80    # HTTP
hcloud firewall rule add openclaw --protocol tcp --port 443   # HTTPS

# 绑定到服务器
hcloud server attach-firewall openclaw --firewall openclaw
```

### 步骤五：SSH 连接并安装

```bash
# 获取 IP
hcloud server ip openclaw

# SSH 连接
ssh root@你的服务器IP

# 安装 Docker
curl -fsSL https://get.docker.com | sh

# 启动 Docker
systemctl start docker
systemctl enable docker
```

### 步骤六：运行 OpenClaw

```bash
# 方式一：Docker 直接运行
docker run -d \
  --name openclaw \
  --restart unless-stopped \
  -p 18789:18789 \
  -v ~/.openclaw:/root/.openclaw \
  -e DEEPSEEK_API_KEY=$DEEPSEEK_API_KEY \
  openclaw/openclaw:latest

# 方式二：使用 docker-setup.sh
git clone https://github.com/openclaw/openclaw.git
cd openclaw
./docker-setup.sh
```

### Hetzner 成本

| 配置 | 价格 |
|-----|------|
| CPX11 (2C/2G) | €4.49/月 |
| 80GB SSD | 已包含 |
| 流量 | 20TB 免费 |
| **总计** | **€4.49/月** |

## 2.9.3 GCP 部署

### 步骤一：创建项目

1. 访问 [Google Cloud Console](https://console.cloud.google.com/)
2. 创建新项目
3. 启用计费

### 步骤二：安装 gcloud CLI

```bash
# macOS
brew install google-cloud-sdk

# 初始化
gcloud init
```

### 步骤三：创建 Compute Engine

```bash
# 创建实例
gcloud compute instances create openclaw \
  --zone=us-central1-a \
  --machine-type=e2-medium \
  --image-family=ubuntu-2204-lts \
  --image-project=ubuntu-os-cloud \
  --boot-disk-size=20GB \
  --tags=http-server,https-server
```

### 步骤四：配置防火墙

```bash
# 开放端口
gcloud compute firewall-rules create openclaw-allow \
  --allow tcp:22,tcp:18789,tcp:80,tcp:443 \
  --source-ranges=0.0.0.0/0
```

### 步骤五：SSH 并安装

```bash
# SSH 连接
gcloud compute ssh openclaw --zone=us-central1-a

# 安装 Docker
curl -fsSL https://get.docker.com | sh

# 运行 OpenClaw
docker run -d \
  --name openclaw \
  --restart unless-stopped \
  -p 18789:18789 \
  openclaw/openclaw:latest
```

### GCP 成本

| 项目 | 价格 |
|-----|------|
| e2-medium | $8.40/月 |
| 20GB SSD | ~$1.70/月 |
| **总计** | **~$10/月** |

## 2.9.4 腾讯云轻量部署（国内推荐）

### 步骤一：购买服务器

1. 登录 [腾讯云轻量应用服务器](https://cloud.tencent.com/lighthouse)
2. 选择 **应用镜像** → **Docker**
3. 选择地域（建议广州/上海）
4. 选择配置（2C2G）
5. 设置登录密码
6. 购买

### 步骤二：配置安全组

```
控制台 → 安全组 → 配置规则

入站规则：
├── TCP 22    (SSH)        ✅
├── TCP 18789  (OpenClaw)  ✅
├── TCP 80     (HTTP)       ✅
└── TCP 443    (HTTPS)      ✅
```

### 步骤三：SSH 连接

```bash
ssh root@你的服务器IP
```

### 步骤四：配置 OpenClaw

```bash
# 创建配置目录
mkdir -p ~/.openclaw

# 编辑配置
vim ~/.openclaw/openclaw.json
```

配置内容：

```json
{
  "agents": {
    "defaults": {
      "model": {
        "provider": "deepseek",
        "model": "deepseek-chat"
      }
    }
  },
  "gateway": {
    "port": 18789
  }
}
```

### 步骤五：运行 OpenClaw

```bash
# 运行容器
docker run -d \
  --name openclaw \
  --restart unless-stopped \
  -p 18789:18789 \
  -v ~/.openclaw:/root/.openclaw \
  -e DEEPSEEK_API_KEY=$DEEPSEEK_API_KEY \
  openclaw/openclaw:latest
```

## 2.9.6 域名与 HTTPS

### 购买域名

```bash
# 阿里云/腾讯云购买
# 建议使用 .com/.cn 常见域名
```

### 配置 DNS 解析

| 记录类型 | 主机记录 | 记录值 |
|---------|---------|--------|
| A | @ | 服务器IP |
| A | www | 服务器IP |

### Nginx 反向代理

```bash
# 安装 Nginx
apt install nginx

# 创建配置
vim /etc/nginx/sites-available/openclaw
```

配置：

```nginx
server {
    listen 80;
    server_name your-domain.com;

    location / {
        proxy_pass http://127.0.0.1:18789;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
}
```

```bash
# 启用配置
ln -s /etc/nginx/sites-available/openclaw /etc/nginx/sites-enabled/
nginx -t
systemctl reload nginx
```

### 配置 HTTPS

#### 方法一：Certbot（推荐）

```bash
# 安装 Certbot
apt install certbot python3-certbot-nginx

# 获取证书
certbot --nginx -d your-domain.com -d www.your-domain.com

# 自动续期
certbot renew --dry-run
```

#### 方法二：手动配置

```nginx
server {
    listen 443 ssl http2;
    server_name your-domain.com;

    ssl_certificate /etc/letsencrypt/live/your-domain.com/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/your-domain.com/privkey.pem;

    location / {
        proxy_pass http://127.0.0.1:18789;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
}
```

## 2.9.7 防火墙配置

### UFW (Ubuntu)

```bash
# 安装 UFW
apt install ufw

# 开放端口
ufw allow 22/tcp    # SSH
ufw allow 80/tcp    # HTTP
ufw allow 443/tcp   # HTTPS
ufw allow 18789/tcp # OpenClaw

# 启用防火墙
ufw enable

# 查看状态
ufw status
```

### firewalld (CentOS)

```bash
# 安装
yum install firewalld

# 开放端口
firewall-cmd --permanent --add-port=22/tcp
firewall-cmd --permanent --add-port=80/tcp
firewall-cmd --permanent --add-port=443/tcp
firewall-cmd --permanent --add-port=18789/tcp

# 重载
firewall-cmd --reload
```

## 2.9.8 监控与维护

### 自动监控脚本

```bash
#!/bin/bash

# 检查 OpenClaw 状态
if ! docker ps | grep -q openclaw; then
    echo "OpenClaw 未运行，正在启动..."
    docker start openclaw
fi

# 检查端口
if ! netstat -tuln | grep -q 18789; then
    echo "端口 18789 未监听"
fi

# 监控资源
docker stats --no-stream openclaw
```

### 定时任务

```bash
# 编辑 crontab
crontab -e

# 每天凌晨 3 点检查
0 3 * * * /path/to/check_openclaw.sh >> /var/log/openclaw.log 2>&1
```
