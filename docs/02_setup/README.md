## 学习目标

完成本章后，你将能够：

- ✅ 了解 OpenClaw 的系统要求
- ✅ 完成本地部署（Docker/Podman/源码）
- ✅ 完成云端部署（腾讯云/Fly.io/Hetzner/GCP）
- ✅ 配置 OpenClaw 并启动运行
- ✅ 使用 Docker 进行容器化部署
- ✅ 连接移动端（iOS/Android）节点

## 前置要求

- Node.js >= 22.0
- 一个 LLM API 密钥
- （可选）云服务器或 Docker
