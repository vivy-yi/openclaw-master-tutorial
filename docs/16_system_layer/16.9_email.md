## 9.2 email_send - 发送邮件

### 功能说明

发送电子邮件。

### 参数

```json5
{
  "to": "user@example.com",          // 收件人
  "cc": "cc@example.com",            // 抄送
  "bcc": "bcc@example.com",          // 密送
  "subject": "邮件主题",             // 主题
  "body": "邮件内容",                // 正文
  "html": "<p>HTML 正文</p>",       // HTML 正文
  "attachments": ["file1.pdf"],      // 附件
  "from": "sender@example.com"       // 发件人（可选）
}
```

### 使用示例

```bash
# 发送简单邮件
email_send({
  to: "user@example.com",
  subject: "Hello",
  body: "这是一封测试邮件"
})

# 发送 HTML 邮件
email_send({
  to: "user@example.com",
  subject: "欢迎注册",
  html: "<h1>欢迎！</h1><p>感谢您的注册。</p>"
})

# 发送带附件的邮件
email_send({
  to: "user@example.com",
  subject: "报告",
  body: "请查收附件",
  attachments: ["/path/to/report.pdf"]
})

# 抄送和密送
email_send({
  to: "user@example.com",
  cc: "manager@example.com",
  bcc: "admin@example.com",
  subject: "项目更新",
  body: "本周项目进度报告"
})
```

## 9.3 email_list - 获取邮件列表

### 功能说明

获取邮箱中的邮件列表。

### 参数

```json5
{
  "folder": "INBOX",       // 文件夹
  "limit": 20,             // 数量限制
  "unread": true,         // 只看未读
  "since": "2024-01-01"   // 时间过滤
}
```

### 使用示例

```bash
# 获取收件箱
email_list({ folder: "INBOX" })

# 获取未读邮件
email_list({ folder: "INBOX", unread: true })

# 获取最近邮件
email_list({ folder: "INBOX", limit: 10 })
```

---

## 9.4 email_read - 读取邮件

### 功能说明

读取指定邮件的详细内容。

### 参数

```json5
{
  "id": "mail-id",        // 邮件 ID
  "downloadAttachments": false  // 是否下载附件
}
```

### 使用示例

```bash
# 读取邮件
email_read({ id: "abc123" })

# 下载附件
email_read({ id: "abc123", downloadAttachments: true })
```

## 9.5 工具配置

### 全局配置

```json5
{
  tools: {
    allow: ["group:email"]
  }
}
```

### 按 Agent 配置

```json5
{
  agents: {
    list: [
      {
        id: "notifier",
        tools: {
          allow: ["email_send"],
          deny: ["email_read", "email_list"]
        }
      }
    ]
  }
}
```

### 邮件服务配置

```json5
{
  agents: {
    defaults: {
      email: {
        // 默认发件人
        defaultFrom: "noreply@example.com",
        // SMTP 配置
        smtp: {
          host: "smtp.example.com",
          port: 587,
          secure: true
        }
      }
    }
  }
}
```
