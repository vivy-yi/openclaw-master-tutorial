## 6.2 calendar_list - 获取日历

### 功能说明

获取用户可用的日历列表。

### 参数

```json5
{
  "account": "primary"  // 可选，日历账户
}
```

### 使用示例

```bash
# 获取所有日历
calendar_list()

# 获取特定账户的日历
calendar_list({ account: "work" })
```

### 返回结果

```json5
{
  "calendars": [
    {
      "id": "primary",
      "name": "主要日历",
      "color": "#4285F4",
      "readOnly": false
    },
    {
      "id": "work",
      "name": "工作日历",
      "color": "#EA4335",
      "readOnly": false
    }
  ]
}
```

## 6.3 calendar_events - 获取日程

### 功能说明

获取指定时间范围内的日程。

### 参数

```json5
{
  "calendar": "primary",       // 日历 ID
  "start": "2024-01-01",      // 开始时间
  "end": "2024-01-31",        // 结束时间
  "maxResults": 50             // 最大结果数
}
```

### 使用示例

```bash
# 获取本周日程
calendar_events({
  calendar: "primary",
  start: "2024-01-01",
  end: "2024-01-07"
})

# 获取今日日程
calendar_events({
  calendar: "primary",
  start: "2024-01-15T00:00:00Z",
  end: "2024-01-15T23:59:59Z"
})
```

### 返回结果

```json5
{
  "events": [
    {
      "id": "abc123",
      "summary": "团队会议",
      "start": {
        "dateTime": "2024-01-15T10:00:00Z",
        "timeZone": "Asia/Shanghai"
      },
      "end": {
        "dateTime": "2024-01-15T11:00:00Z",
        "timeZone": "Asia/Shanghai"
      },
      "attendees": [
        { "email": "user@example.com" }
      ]
    }
  ]
}
```

## 6.4 calendar_create - 创建日程

### 功能说明

创建新的日历日程。

### 参数

```json5
{
  "calendar": "primary",       // 日历 ID
  "summary": "会议名称",       // 标题
  "description": "会议描述",   // 描述
  "start": "2024-01-15T10:00:00Z",  // 开始时间
  "end": "2024-01-15T11:00:00Z",    // 结束时间
  "attendees": ["user@example.com"], // 参与者
  "location": "会议室A",       // 地点
  "reminders": {               // 提醒
    "useDefault": false,
    "overrides": [
      { "method": "email", "minutes": 30 },
      { "method": "popup", "minutes": 10 }
    ]
  }
}
```

### 使用示例

```bash
# 创建简单日程
calendar_create({
  calendar: "primary",
  summary: "周例会",
  start: "2024-01-15T10:00:00Z",
  end: "2024-01-15T11:00:00Z"
})

# 创建全天事件
calendar_create({
  calendar: "primary",
  summary: "假期",
  start: { "date": "2024-01-20" },
  end: { "date": "2024-01-21" }
})

# 创建会议并邀请参与者
calendar_create({
  calendar: "work",
  summary: "项目评审会",
  description: "评审项目进度",
  start: "2024-01-15T14:00:00Z",
  end: "2024-01-15T15:30:00Z",
  location: "会议室B",
  attendees: [
    "user1@example.com",
    "user2@example.com"
  ],
  reminders: {
    useDefault: false,
    overrides: [
      { "method": "email", "minutes": 60 },
      { "method": "popup", "minutes": 15 }
    ]
  }
})
```

## 6.5 calendar_update - 更新日程

### 功能说明

更新已有的日历日程。

### 参数

```json5
{
  "calendar": "primary",       // 日历 ID
  "eventId": "abc123",         // 日程 ID
  "summary": "新标题",          // 新标题
  "description": "新描述",      // 新描述
  "start": "2024-01-15T11:00:00Z",  // 新开始时间
  "end": "2024-01-15T12:00:00Z"     // 新结束时间
}
```

### 使用示例

```bash
# 更新标题
calendar_update({
  calendar: "primary",
  eventId: "abc123",
  summary: "新会议名称"
})

# 更新时间和参与者
calendar_update({
  calendar: "primary",
  eventId: "abc123",
  start: "2024-01-15T15:00:00Z",
  end: "2024-01-15T16:00:00Z",
  attendees: ["new-user@example.com"]
})
```

## 6.7 工具配置

### 全局配置

```json5
{
  tools: {
    allow: ["group:calendar"]
  }
}
```

### 按 Agent 配置

```json5
{
  agents: {
    list: [
      {
        id: "assistant",
        tools: {
          allow: ["group:calendar"],
          deny: ["calendar_delete"]  // 只读
        }
      }
    ]
  }
}
```

### 日历账户配置

```json5
{
  agents: {
    defaults: {
      calendar: {
        // 默认日历
        defaultCalendar: "primary",
        // 允许访问的日历
        allowedCalendars: ["primary", "work"],
        // 时区
        timeZone: "Asia/Shanghai"
      }
    }
  }
}
```

## 6.9 最佳实践

### 智能日程创建

```bash
# 检查冲突后创建
events = calendar_events({
  calendar: "primary",
  start: "2024-01-15T10:00:00Z",
  end: "2024-01-15T12:00:00Z"
})

if (events.length === 0) {
  calendar_create({
    calendar: "primary",
    summary: "新会议",
    start: "2024-01-15T10:00:00Z",
    end: "2024-01-15T11:00:00Z"
  })
} else {
  print("时间冲突")
}
```

### 自动提醒

```bash
# 创建带提醒的日程
calendar_create({
  calendar: "primary",
  summary: "重要会议",
  start: "2024-01-15T10:00:00Z",
  end: "2024-01-15T11:00:00Z",
  reminders: {
    useDefault: false,
    overrides: [
      { "method": "email", "minutes": 60 },  // 1小时前邮件
      { "method": "popup", "minutes": 15 }   // 15分钟前弹窗
    ]
  }
})
```
