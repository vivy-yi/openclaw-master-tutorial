## 4.2 fetch - HTTP 请求

### 功能说明

发送 HTTP 请求，支持 GET、POST、PUT、DELETE 等方法。

### 参数

```json5
{
  "url": "https://api.example.com/data",  // 必需，请求 URL
  "method": "GET",                         // 请求方法
  "headers": {                             // 请求头
    "Authorization": "Bearer token",
    "Content-Type": "application/json"
  },
  "body": {},                              // 请求体
  "timeout": 30000                         // 超时时间
}
```

### 使用示例

#### GET 请求

```bash
# 简单 GET 请求
fetch("https://api.example.com/users")

# 带查询参数
fetch("https://api.example.com/users", {
  method: "GET",
  query: {
    page: 1,
    limit: 10
  }
})
```

#### POST 请求

```bash
# POST JSON
fetch("https://api.example.com/users", {
  method: "POST",
  headers: {
    "Content-Type": "application/json"
  },
  body: {
    name: "张三",
    email: "zhangsan@example.com"
  }
})

# POST 表单
fetch("https://api.example.com/login", {
  method: "POST",
  headers: {
    "Content-Type": "application/x-www-form-urlencoded"
  },
  body: "username=admin&password=123456"
})
```

#### PUT/DELETE 请求

```bash
# PUT 更新
fetch("https://api.example.com/users/1", {
  method: "PUT",
  body: { name: "新名字" }
})

# DELETE 删除
fetch("https://api.example.com/users/1", {
  method: "DELETE"
})
```

## 4.3 响应处理

### 响应结构

```json5
{
  "status": 200,
  "statusText": "OK",
  "headers": {
    "content-type": "application/json"
  },
  "body": {
    "data": [...]
  },
  "error": null
}
```

### 处理响应

```bash
response = fetch("https://api.example.com/users/1")

if (response.status === 200) {
  user = response.body
  print("User: " + user.name)
} else {
  print("Error: " + response.error)
}
```

## 4.5 Webhook 配置

### 功能说明

配置 Webhook 用于接收外部请求。

### 参数

```json5
{
  "path": "/webhook/github",     // Webhook 路径
  "method": "POST",              // 支持的方法
  "handler": "handleWebhook",    // 处理函数
  "secret": "your-secret"        // 验证密钥
}
```

### 使用示例

```bash
# 配置 Webhook
webhook({
  path: "/webhook/github",
  method: "POST",
  handler: "handleGitHubEvent"
})

# 处理 GitHub Webhook
function handleGitHubEvent(request) {
  event = request.headers["x-github-event"]
  payload = request.body

  if (event === "push") {
    print("New push: " + payload.ref)
  } else if (event === "pull_request") {
    print("PR: " + payload.action)
  }

  return { status: 200 }
}
```

```json5

  }

```

```json5
{
  agents: {
    list: [
      {
        id: "api-client",
        tools: {
          allow: ["group:http"],
          deny: []  // 完全允许
        }
      }
    ]
  }
}
```

## 4.7 重试机制

### 自动重试配置

```json5
{
  fetch: {
    retry: {
      maxAttempts: 3,
      delay: 1000,
      backoff: "exponential",
      retryOn: [408, 429, 500, 502, 503, 504]
    }
  }
}
```

### 手动重试

```bash
function fetchWithRetry(url, options, maxRetries = 3) {
  for (i = 0; i < maxRetries; i++) {
    try {
      response = fetch(url, options)
      if (response.status < 500) {
        return response
      }
    } catch (error) {
      print("Attempt " + (i + 1) + " failed: " + error.message)
    }
    sleep(1000 * (i + 1))
  }
  throw new Error("Max retries exceeded")
}
```

## 4.9 最佳实践

### 基础 URL 配置

```json5
{
  agents: {
    defaults: {
      fetch: {
        baseURL: "https://api.example.com",
        defaultHeaders: {
          "User-Agent": "OpenClaw/1.0"
        }
      }
    }
  }
}
```

### 统一错误处理

```bash
async function apiRequest(url, options) {
  try {
    response = await fetch(url, options)

    if (response.status >= 400) {
      throw new Error(`API Error: ${response.status} - ${response.error}`)
    }

    return response.body
  } catch (error) {
    print("Request failed: " + error.message)
    throw error
  }
}
```
