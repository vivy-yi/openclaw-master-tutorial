| 方式 | 智能机器人 | 自建应用 |
|------|-----------|---------|
| 连接方式 | WebSocket 长连接 | HTTPS 回调 |
| 主动发送 | 受限 | 支持 |
| 群聊支持 | ✅ | ❌ |
| 配置复杂度 | 简单 | 中等 |
| 需要公网 | 否 | 是 |

## 3.3 企业微信开放平台配置

### 创建智能机器人

1. 登录[企业微信管理后台](https://work.weixin.qq.com/wework_admin/)
2. 进入「应用管理」→「智能机器人」
3. 创建机器人，选择**长连接模式**
4. 记录 `BotId` 和 `Secret`

### 创建自建应用（可选）

如需主动发送消息，创建自建应用：

1. 在企业微信后台创建自建应用
2. 记录 `CorpID`、`AgentId`、`Secret`
3. 配置可信 IP

### 必需权限

```
智能机器人权限：
├── im:message - 消息收发
├── im:resource - 媒体文件
└── contact:user - 用户信息

自建应用额外权限：
├── message:send - 主动发送消息
├── media:upload - 文件上传
└── user:get - 用户获取
```

## 3.4 配置

### 智能机器人配置

```bash
# 启用插件
openclaw config set plugins.entries.wecom.enabled true

# 配置渠道
openclaw config set channels.wecom.enabled true
openclaw config set channels.wecom.botId your-bot-id
openclaw config set channels.wecom.secret your-bot-secret
openclaw config set channels.wecom.dmPolicy pairing
openclaw config set channels.wecom.groupPolicy open
```

### 完整配置示例

```json
{
  "plugins": {
    "entries": {
      "wecom": {
        "enabled": true
      }
    }
  },
  "channels": {
    "wecom": {
      "enabled": true,
      "botId": "aibxxxxxxxxxxxxxxxx",
      "secret": "xxxxxxxxxxxxxxxx",
      "welcomeMessage": "你好，我是 AI 助手。",
      "sendThinkingMessage": true,
      "dmPolicy": "pairing",
      "allowFrom": [],
      "groupPolicy": "open",
      "groupAllowFrom": [],
      "adminUsers": [],
      "commands": {
        "enabled": true,
        "allowlist": ["/new", "/compact", "/help", "/status"]
      },
      "dynamicAgents": {
        "enabled": true
      },
      "groupChat": {
        "enabled": true,
        "requireMention": true
      }
    }
  }
}
```

### 配置选项

| 选项 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `enabled` | boolean | true | 是否启用 |
| `botId` | string | 必填 | 机器人 Bot ID |
| `secret` | string | 必填 | 机器人密钥 |
| `dmPolicy` | string | "pairing" | 私聊策略 |
| `groupPolicy` | string | "open" | 群聊策略 |
| `welcomeMessage` | string | - | 入群欢迎语 |
| `sendThinkingMessage` | boolean | true | 是否发送思考占位 |

### 私聊策略 (dmPolicy)

| 值 | 说明 |
|---|------|
| `pairing` | 默认模式，需要配对码 |
| `allowlist` | 仅白名单用户 |
| `open` | 完全开放 |
| `disabled` | 禁用私聊 |

### 群聊策略

| 配置 | 说明 |
|------|------|
| `groupPolicy: "open"` | 所有群可触发 |
| `groupPolicy: "allowlist"` + `groupAllowFrom` | 只允许指定群 |
| `groupPolicy: "disabled"` | 禁用群聊 |
| `requireMention` | 是否需要 @ 才响应 |

## 3.7 动态 Agent

### 工作原理

- 私聊：`wecom-dm-<userId>`
- 群聊：`wecom-group-<chatId>`

每个用户/群聊独立上下文和记忆。

### 配置示例

```json
{
  "channels": {
    "wecom": {
      "dynamicAgents": {
        "enabled": true,
        "adminBypass": false
      },
      "dm": {
        "createAgentOnFirstMessage": true
      },
      "workspaceTemplate": "/path/to/template-dir"
    }
  }
}
```

### Workspace 模板

`workspaceTemplate` 目录中的文件会在动态 Agent 首次创建时复制：
- `AGENTS.md`
- `BOOTSTRAP.md`
- `CLAUDE.md`
- `SOUL.md`
- `TOOLS.md`
- `IDENTITY.md`
- `USER.md`
- `HEARTBEAT.md`

## 3.9 故障排除

### 常见问题

**Q: 消息收不到**
1. 检查机器人是否已切换到长连接模式
2. 确认应用已发布
3. 检查 Gateway 日志

**Q: 权限不足**
1. 检查应用权限配置
2. 确认已发布应用
3. 重新安装应用到群

**Q: 60020 not allow to access**
- 企业微信自建应用 API 的可信 IP 限制
- 把服务器出口 IP 加入可信 IP 白名单

## 本节小结

1. **智能机器人**：WebSocket 长连接，无需公网 IP
2. **自建应用**：HTTPS 回调，支持主动发送
3. **消息**：支持文本/图片/语音/文件
4. **动态 Agent**：每个用户/群聊独立上下文
5. **多账号**：支持多机器人配置
