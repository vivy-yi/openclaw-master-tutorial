# 11.4 审计日志

## 本节目标
- 掌握审计配置
- 理解日志内容
- 完成审计集成

---

## 11.4.1 审计概述

### 记录内容

```
审计日志内容：

1. 身份验证
   ├── 登录/登出
   ├── 认证失败
   └── 权限变更

2. 操作记录
   ├── API 调用
   ├── 工具执行
   └── 配置变更

3. 敏感操作
   ├── 文件访问
   ├── 命令执行
   └── 数据导出
```

---

## 11.4.2 配置审计

### 基础配置

```yaml
security:
  audit:
    enabled: true
    retention: 90d

    # 记录内容
    events:
      - auth
      - api_calls
      - tool_executions
      - config_changes
      - file_access

    # 输出
    outputs:
      - type: file
        path: ./logs/audit.log
      - type: syslog
        server: syslog.example.com
```

```json
{

    "user_agent": "OpenClaw/1.0"

```

---

## 11.4.4 查询分析

### 查询示例

```bash
# 查看用户操作
openclaw audit query --user user_123

# 查看失败操作
openclaw audit query --status failed

# 查看敏感操作
openclaw audit query --event file_access

# 时间范围
openclaw audit query --from 2024-01-01 --to 2024-01-15
```
