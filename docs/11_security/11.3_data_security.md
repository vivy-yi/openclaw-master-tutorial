# 11.3 数据安全

## 本节目标
- 掌握数据加密配置
- 理解敏感信息保护
- 完成数据安全配置

---

## 11.3.1 加密配置

### 存储加密

```yaml
security:
  encryption:
    enabled: true
    algorithm: AES-256-GCM
    key_rotation:
      enabled: true
      interval: 90d
```

### 传输加密

```yaml
security:
  tls:
    enabled: true
    cert: /path/to/cert.pem
    key: /path/to/key.pem
    min_version: TLSv1.2
```

---

## 11.3.2 敏感信息

### 敏感字段

```
敏感信息：

1. 认证信息
   ├── API Keys
   ├── Passwords
   └── Tokens

2. 用户数据
   ├── 个人隐私
   └── 聊天记录

3. 业务数据
   ├── API 使用量
   └── 成本信息
```

### 保护配置

```yaml
security:
  sensitive_data:
    detection:
      enabled: true
      patterns:
        - "sk-.*"
        - "password.*"
        - "token.*"

    masking:
      enabled: true
      fields:
        - api_key
        - password
        - token
```

---

## 11.3.3 数据脱敏

### 脱敏规则

```yaml
security:
  masking:
    rules:
      - field: api_key
        pattern: "(sk-).{10,}"
        replacement: "sk-***"

      - field: email
        pattern: "(.*)@(.*)"
        replacement: "***@***"

      - field: phone
        pattern: "(\\d{3})\\d{4}(\\d{4})"
        replacement: "$1****$2"
```
