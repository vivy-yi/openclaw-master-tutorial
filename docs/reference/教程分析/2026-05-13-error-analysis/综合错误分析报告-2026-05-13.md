## ✅ 正确的架构（官方参考）

### 记忆系统
```
workspace/
├── MEMORY.md              # 长期记忆
├── memory/
│   └── YYYY-MM-DD.md      # 每日日志
└── DREAMS.md              # 梦境日记（可选）
```

### Agent 架构
- Workspace: `~/.openclaw/workspace` 或 `~/.openclaw/workspace-<agentId>`
- 多 Agent: 使用 bindings 配置 + sessions_send/sessions_spawn
- 没有 "Agent Dispatcher"、"Result Aggregator"、"teams" 等组件

### Skills 架构
- 只需 SKILL.md 一个文件
- 调用语法: `/skill-name` (slash command)
- 加载位置: workspace > personal > managed > bundled

### Channel 架构
- 配置路径: 顶层 `channels` (不是 `gateway.channels`)
- Token 引用: 通过 SecretRef (env/file/exec)
- 飞书/Discord: 内置，无需安装插件
