## 二、平台详细信息

### 2.1 小红书

**平台类型**: 社交种草平台

**数据价值**:
- 竞品营销策略
- KOL/KOC 合作
- 用户口碑分析
- 热门话题趋势

**采集方式**:
```yaml
API端点: https://edith.xiaohongshu.com/api/sns/web/v2/user/profile
拦截模式: browser.intercept()
关键数据:
  - 用户信息（粉丝、获赞、收藏）
  - 笔记列表
  - 评论数据
  - 话题标签
```

**配置文件**:
```yaml
xiaohongshu:
  name: "小红书"
  icon: "📕"
  url_pattern: "xiaohongshu.com/user/profile/*"
  selectors:
    name: ".user-name"
    followers: ".user-info-item:nth-child(1) .count"
    posts: ".user-info-item:nth-child(3) .count"
```

## 六、配置示例

### 添加新竞品

```yaml
# 在 profiles/competitors/config.yaml 中添加
- name: "新竞品"
  enabled: true
  platforms:
    xiaohongshu:
      url: "https://www.xiaohongshu.com/user/profile/竞品ID"
    zhihu:
      url: "https://www.zhihu.com/people/竞品ID"
```

### 添加新话题

```yaml
# 在 profiles/aggregated/config.yaml 中添加
- name: "新话题"
  enabled: true
  keywords:
    - "关键词1"
    - "关键词2"
  platforms:
    zhihu:
      url: "https://www.zhihu.com/search?type=content&q=关键词"
```

### 添加新平台到画像

```yaml
# 在 profiles/users/config.yaml 中添加
platforms:
  new_platform:
    enabled: true
    url: "https://platform.com/profile"
    type: "social"
```
