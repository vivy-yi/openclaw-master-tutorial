## 14.10.1 n8n工作流编排

### 场景描述

通过Webhook委托n8n工作流，代理不直接接触凭据。

### 架构

```
OpenClaw → Webhook → n8n → 第三方API
              ↓
         凭据在n8n端
```

### 配置

```json5
{
  "webhooks": {
    "n8n": {
      "url": "https://n8n.example.com/webhook/my-workflow",
      "method": "POST"
    }
  }
}
```

## 14.10.2 自我修复家庭服务器

### 功能

- SSH访问
- 定时健康检查
- 自动重启服务
- 日志监控

### 配置

```bash
openclaw cron add \
  --name "Self Heal" \
  --cron "*/15 * * * *" \
  --message "健康检查：
1. ping检查
2. 服务状态
3. 磁盘空间
4. 内存使用

异常自动尝试修复" \
  --announce
```
