## 14.4.1 三层记忆系统

### 架构

```
┌─────────────────────────────────────────────┐
│              三层记忆系统                      │
├─────────────────────────────────────────────┤
│                                             │
│  长期记忆 (Long-term)                        │
│  ├── 重要联系人信息                           │
│  ├── 项目背景知识                            │
│  └── 偏好和习惯                              │
│                                             │
│  每日记忆 (Daily)                           │
│  ├── 今日总结                               │
│  ├── 关键决策                              │
│  └── 待办跟进                               │
│                                             │
│  工作记忆 (Working)                          │
│  ├── 当前任务上下文                          │
│  └── 实时对话                               │
│                                             │
└─────────────────────────────────────────────┘
```

### 配置

```json5
{
  "memory": {
    "tiers": {
      "longTerm": { "retention": "forever" },
      "daily": { "retention": "30d", "compress": true },
      "working": { "retention": "session" }
    }
  }
}
```

## 14.4.2 知识图谱重建

### 场景描述

夜间自动重建知识图谱，保持信息关联。

```bash
openclaw cron add \
  --name "Knowledge Graph Rebuild" \
  --cron "0 2 * * *" \
  --message "重建知识图谱：
1. 扫描所有新文档
2. 提取实体和关系
3. 更新图数据库
4. 报告新增连接" \
  --no-deliver
```

## 14.4.3 早间摘要生成

### 场景描述

每天早上生成夜间活动摘要。

```bash
openclaw cron add \
  --name "Morning Digest" \
  --cron "0 7 * * *" \
  --message "生成昨夜摘要：
1. Cron任务执行结果
2. 收到的消息统计
3. 重要事件回顾
4. 今日建议关注" \
  --announce
```
