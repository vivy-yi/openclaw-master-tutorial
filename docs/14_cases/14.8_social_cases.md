## 14.8.1 Reddit每日摘要

### 配置

```bash
openclaw cron add \
  --name "Reddit Digest" \
  --cron "0 8 * * *" \
  --message "生成Reddit摘要：
1. 获取配置的Subreddit热门
2. 按关键词过滤
3. 选取最佳内容
4. 生成摘要" \
  --announce
```

## 14.8.2 YouTube每日摘要

## 14.8.3 X账户分析

### 场景描述

# 分析指定账户
