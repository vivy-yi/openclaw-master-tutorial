## 14.6.1 GitHub Issue优先级排序

### 配置

```bash
openclaw cron add \
  --name "Issue Prioritizer" \
  --cron "0 9 * * 1" \
  --message "排序GitHub Issue：
1. 按紧急程度分级
2. 识别阻塞项
3. 建议处理顺序
4. 分配负责人" \
  --announce
```

## 14.6.3 链上钱包监控

### 场景描述

监控加密钱包余额变化。

```bash
openclaw cron add \
  --name "Wallet Monitor" \
  --cron "*/30 * * * *" \
  --message "监控钱包：
1. 检查余额变化
2. 大额转入/转出告警
3. Gas费用分析
4. 生成日报" \
  --announce
```
