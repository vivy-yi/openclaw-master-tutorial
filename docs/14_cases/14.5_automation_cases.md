## 14.5.1 凌晨5点基础设施健康检查

### 场景描述

每天早上5点检查服务器和服务健康状态。

```bash
openclaw cron add \
  --name "Morning Health Check" \
  --cron "0 5 * * *" \
  --tz "Asia/Shanghai" \
  --session isolated \
  --message "执行健康检查：

1. 服务器ping延迟
2. 关键服务状态（nginx, docker, mysql）
3. 磁盘使用率
4. 内存和CPU
5. 最近错误日志

异常列出问题和建议，正常报告'全部正常'" \
  --announce
```

## 14.5.2 交易机器人监控

### 场景描述

监控交易机器人状态，异常时自动重启。

```bash
openclaw cron add \
  --name "Trading Bot Monitor" \
  --cron "*/15 * * * *" \
  --message "检查交易机器人：
1. 进程状态
2. API连接
3. 持仓状态
4. 错误日志

异常执行：重启并恢复数据" \
  --announce
```
