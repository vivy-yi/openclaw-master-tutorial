## 5.6.1 内置 Skills 概述

### 什么是内置 Skills

```
┌─────────────────────────────────────────────────────────────┐
│                    内置 Skills                               │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  OpenClaw 开箱即用的 Skills：                              │
```

## 5.6.2 开发类 Skills

### github - GitHub 操作

**功能**：通过 `gh` CLI 操作 GitHub 仓库、Issues、PRs、CI

**使用场景**：
- 查看 PR 状态和 CI 结果
- 创建/评论/关闭 Issues
- 查看/创建/合并 Pull Requests
- 查询 GitHub API

**前置要求**：
```bash
# 安装 GitHub CLI
brew install gh

# 认证
gh auth login
```

**常用命令**：
```bash
# 列出 PRs
gh pr list --repo owner/repo

# 查看 CI 状态
gh pr checks 55 --repo owner/repo

# 创建 Issue
gh issue create --title "Bug: xxx" --body "描述"

# 查看 CI 运行
gh run list --repo owner/repo --limit 10

# 重新运行失败的 job
gh run rerun <run-id> --failed --repo owner/repo
```

**在对话中使用**：
```
用户：检查一下这个 PR 的 CI 状态
→ 自动使用 github skill
← 返回 CI 检查结果
```

---

### coding-agent - 代码开发

**功能**：完整的代码开发能力，包括读取、编辑、测试、调试

**使用场景**：
- 新功能开发
- Bug 修复
- 代码重构
- 测试编写

**特点**：
- 可使用文件系统工具

- 支持多文件操作

**在对话中使用**：
```
用户：帮我实现一个用户认证模块
→ 使用 coding-agent skill
← 返回完整实现代码
```

## 5.6.3 笔记类 Skills

### obsidian - Obsidian 笔记

**功能**：操作 Obsidian 笔记库

**使用场景**：
- 读取笔记
- 搜索内容
- 创建/更新笔记
- 管理金丝雀发布

**前置要求**：
```bash
# Obsidian Vault 路径
# 配置中指定 vault 路径
```

**常用操作**：
```bash
# 读取笔记
obsidian read "我的笔记"

# 搜索
obsidian search "关键字"

# 创建笔记
obsidian create "新笔记.md" --content "# 内容"
```

---

### notion - Notion 集成

**功能**：操作 Notion 数据库和页面

**使用场景**：
- 读取 Notion 页面
- 创建数据库条目
- 更新任务状态

**前置要求**：
- Notion API Key
- 数据库 ID

**配置**：
```bash
openclaw config set plugins.entries.notion.config.apiKey "xxx"
openclaw config set plugins.entries.notion.config.databaseId "xxx"
```

## 5.6.8 调用内置 Skills

### 自动调用

```
用户：帮我检查一下 GitHub PR 的状态
→ 自动识别需要使用 github skill
→ 执行相关命令
← 返回结果
```

### 手动指定

```
用户：使用 github skill 查看最近的 issues
→ 明确指定使用 github skill
← 返回 Issues 列表
```

### 查看可用 Skills

```bash
# 列出所有可用 Skills
openclaw skills list

# 查看 Skills 状态
openclaw status
```

## 5.6.9 Skills 配置

### 启用/禁用 Skills

```json
{
  "skills": {
    "enabled": true,
    "autoLoad": true
  }
}
```

### 配置 Skills 路径

```json
{
  "skills": {
    "paths": [
      "~/.openclaw/skills",
      "./custom-skills"
    ]
  }
}
```
