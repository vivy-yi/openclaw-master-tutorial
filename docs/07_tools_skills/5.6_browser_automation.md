## 核心概念

```
OpenClaw 浏览器 = 独立Profile + CDP控制 + Playwright增强 + 确定性Tab控制
```

- **独立 Profile**：OpenClaw 管理的独立 Chrome 实例，与个人浏览器隔离
- **CDP 控制**：通过 Chrome DevTools Protocol 控制浏览器
- **Playwright 增强**：高级操作（点击/输入/快照/PDF）需要 Playwright
- **确定性 Tab 控制**：通过 targetId 精确定位 Tab，而非"最后一个Tab"
