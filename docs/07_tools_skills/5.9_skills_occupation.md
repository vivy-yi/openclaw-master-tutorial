## 5.9.2 金融投资场景

### 股票分析

**推荐 Skills**：

| Skill | 功能 | 说明 |
|-------|------|------|
| `finance-data` | 金融数据查询 | 股票、基金、财报 |
| `stock-analysis` | 股票分析 | 技术分析、基本面 |
| `market-news` | 市场新闻 | 实时资讯 |

**使用示例**：
```
用户：分析一下苹果公司的股票
→ 获取 AAPL 股票数据
→ 分析技术指标
→ 查询最新财报
← 返回分析报告：
  当前价格: $175.50
  市盈率: 28.5
  建议: 持有
```

**配置金融数据**：
```bash
# 配置数据源
openclaw config set plugins.entries.finance.config.apiKey "xxx"
```

**使用示例**：
```
用户：写一个查询用户活跃度的 SQL
→ 分析需求
→ 生成 SQL
← 返回：
```sql
SELECT
  DATE_FORMAT(login_time, '%Y-%m-%d') as date,
  COUNT(DISTINCT user_id) as dau,
  COUNT(*) as login_count
FROM user_login
WHERE login_time >= DATE_SUB(NOW(), INTERVAL 30 DAY)
GROUP BY DATE_FORMAT(login_time, '%Y-%m-%d')
ORDER BY date;
```
