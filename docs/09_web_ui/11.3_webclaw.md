## 3.1 工具简介

**webclaw** 是一个轻量级的 OpenClaw Web 客户端，提供快速访问 OpenClaw 的界面。

### 核心特点

- 快速部署
- 简洁界面
- 跨平台访问

---

## 3.2 快速部署

### 使用 npx 启动

```bash
npx webclaw
```

CLI 会提示输入：
- 项目名称
- 环境变量
- 端口号

### 环境变量

```bash
# 方式一：使用 URL + Token
CLAWDBOT_GATEWAY_URL=http://localhost:18789
CLAWDBOT_GATEWAY_TOKEN=your-token

# 方式二：使用 URL + 密码
CLAWDBOT_GATEWAY_URL=http://localhost:18789
CLAWDBOT_GATEWAY_PASSWORD=your-password
```

### 完整启动示例

```bash
# 设置环境变量
export CLAWDBOT_GATEWAY_URL=http://localhost:18789
export CLAWDBOT_GATEWAY_TOKEN=your-token

# 启动
npx webclaw
```

---

## 3.3 Docker 部署

```bash
docker run -d \
  -p 8080:8080 \
  -e CLAWDBOT_GATEWAY_URL=http://host:18789 \
  -e CLAWDBOT_GATEWAY_TOKEN=your-token \
  ibelick/webclaw
```
