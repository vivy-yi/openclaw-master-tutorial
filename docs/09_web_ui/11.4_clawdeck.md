## 4.1 工具简介

**ClawDeck** 是一个开源的 Agent 任务控制中心，使用 Kanban 风格管理 AI Agent 任务。

### 核心功能

- **Kanban 看板** - 多看板任务组织
- **任务分配** - 分配任务给 Agent
- **活动流** - 实时查看 Agent 工作状态
- **API 访问** - 完整的 REST API
- **实时更新** - Hotwire 驱动实时 UI

### 任务状态

| 状态 | 说明 |
|------|------|
| inbox | 新任务，未分配 |
| up_next | 准备分配 |
| in_progress | 进行中 |
| in_review | 待审核 |
| done | 已完成 |

## 4.2 快速开始

### 方式一：使用托管平台

访问 [clawdeck.io](https://clawdeck.io) 注册使用

### 方式二：自托管部署

```bash
# 克隆仓库
git clone https://github.com/clawdeckio/clawdeck.git
cd clawdeck

# 安装依赖
bundle install

# 初始化数据库
bin/rails db:prepare

# 启动
bin/dev
```

访问 `http://localhost:3000`

## 4.4 API 集成

### 认证

```bash
# 请求头
Authorization: Bearer YOUR_TOKEN
X-Agent-Name: Maxie
X-Agent-Emoji: 🦊
```

### 任务 API

```bash
# 获取分配给自己的任务
GET /api/v1/tasks?assigned=true

# 创建任务
POST /api/v1/tasks
{
  "name": "Research topic X",
  "status": "inbox",
  "board_id": 1
}

# 更新任务状态
PATCH /api/v1/tasks/:id
{
  "status": "in_progress",
  "activity_note": "Starting work"
}

# 完成任务
PATCH /api/v1/tasks/:id/complete
```

### Agent 端集成示例

```python
import requests

# 获取待办任务
def get_my_tasks():
    response = requests.get(
        "https://your-clawdeck.com/api/v1/tasks",
        params={"assigned": "true"},
        headers={
            "Authorization": "Bearer YOUR_TOKEN",
            "X-Agent-Name": "Maxie",
            "X-Agent-Emoji": "🦊"
        }
    )
    return response.json()

# 更新任务状态
def update_task(task_id, status, note=""):
    response = requests.patch(
        f"https://your-clawdeck.com/api/v1/tasks/{task_id}",
        json={"status": status, "activity_note": note},
        headers={"Authorization": "Bearer YOUR_TOKEN"}
    )
    return response.json()
```
