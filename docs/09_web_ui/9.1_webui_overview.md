## 1.4 安装方式

### Docker 部署（推荐）

```bash
# Star-Office-UI
docker run -d -p 19000:19000 ringhyacinth/star-office-ui

# webclaw
docker run -d -p 8080:8080 ibelick/webclaw
```

### 手动部署

```bash
# 克隆仓库
git clone https://github.com/ringhyacinth/Star-Office-UI.git
cd Star-Office-UI

# 安装依赖
pip install -r backend/requirements.txt

# 启动
python3 backend/app.py
```
