## 3.2.1 对话方式

### CLI 对话

```bash
# 启动对话
openclaw chat

# 或使用 gateway
openclaw gateway start

# 然后在 Web UI 中对话
```

### Web UI 对话

访问：`http://127.0.0.1:18789/`

### IM 平台对话

接入飞书/Telegram 后可直接在对应平台对话。

## 3.2.5 自然语言理解

### 模糊指令

```
用户: 那个文件
```

OpenClaw 会：
1. 尝试理解"那个文件"指什么
2. 根据上下文推断
3. 如果无法确定，会询问确认

### 明确指令

```
用户: 打开桌面上最新的PDF文件
```

OpenClaw 会：
1. 搜索桌面 PDF 文件
2. 按修改时间排序
3. 打开最新的文件

## 3.2.6 对话上下文

### 单轮对话

```
用户: 你好
OpenClaw: 你好！
```

### 多轮对话

```
用户: 帮我找一下发票
OpenClaw: 找到以下发票文件：
  - 2024-01-发票.pdf
  - 2024-02-发票.pdf
  - 2023-12-发票.pdf

用户: 打开最新的那个

```

### 上下文记忆

```
用户: 翻译成英文
OpenClaw: [基于上一轮文档内容翻译]
```

## 3.2.7 错误处理

### 权限不足

```
用户: 执行 sudo rm -rf /

OpenClaw: ⚠️ 抱歉，我不能执行这个命令。
这是一个危险操作，可能会导致系统损坏。
如果你确实需要删除文件，请告诉我具体要删除的文件路径。
```

### 文件不存在

```
用户: 打开不存在的文件.txt

OpenClaw: ⚠️ 文件 "不存在的文件.txt" 不存在。

可能的原因：
- 文件名拼写错误
- 文件已被删除或移动
- 不在该目录下

需要我帮你搜索这个文件吗？
```

### API 错误

```
OpenClaw: ⚠️ 遇到了 API 错误。

错误信息：Rate limit exceeded
请稍后重试，或更换 API Key。
```
