## 3.5.1 每日记忆概述

### 什么是每日记忆

每日记忆是 OpenClaw 的短期记忆系统，存储在 `workspace/daily/` 目录下的日期文件中。它记录了每天的会话内容、重要事项和学习新知。

```

│                            每日记忆架构                                         │
├─────────────────────────────────────────────────────────────────────────────────┤
│                                                                                 │
│   ┌─────────────────────────────────────────────────────────────────────┐   │
│   │                       workspace/daily/                               │   │
│   │   ┌─────────────┐  ┌─────────────┐  ┌─────────────┐              │   │
│   │   │ 2024-01-15.md│  │ 2024-01-16.md│  │ 2024-01-17.md│   ...     │   │
│   │   ├─────────────┤  ├─────────────┤  ├─────────────┤              │   │
│   │   │ 今日事项    │  │ 今日事项    │  │ 今日事项    │              │   │
│   │   │ 学习新知    │  │ 学习新知    │  │ 学习新知    │              │   │
│   │   │ 待办事项    │  │ 待办事项    │  │ 待办事项    │              │   │

│                                                                                 │
│   特点：                                                                       │
│   • 按日期自动创建                                                            │

│   • 会话结束自动保存                                                          │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘
```

### 每日记忆 vs 长期记忆

| 特性 | 长期记忆 (MEMORY.md) | 每日记忆 (daily/) |
|-----|---------------------|------------------|
| 存储位置 | `memory/MEMORY.md` | `daily/YYYY-MM-DD.md` |
| 内容周期 | 长期有效 | 当天有效 |
| 更新频率 | 低 | 高 |
| 加载时机 | 每次会话 | 每次会话 |
| 内容类型 | 核心知识、规则、偏好 | 今日事项、学习新知 |

## 3.5.2 记忆文件结构

### 每日文件标准格式

```markdown
# 2024-01-15 每日记忆

## 今日事项
- 完成了用户认证模块的开发
- 修复了登录页面的 bug
- 与产品经理讨论了下迭代需求

## 学习新知
- 用户偏好使用 React Hooks
- 用户公司使用 PostgreSQL 数据库
- 用户提到了想要尝试 TypeScript

## 待办事项
- [ ] 代码审查
- [ ] 更新 API 文档
- [ ] 准备下周一的技术分享

## 重要信息
- 用户的 API 文档放在 ~/docs/api.md
- 下周三之前需要完成支付模块

## 会话摘要
- 会话 1: 讨论了认证流程实现
- 会话 2: 修复了 bug
- 会话 3: 规划了下迭代功能
```

### 自动记录内容

```
每日记忆自动记录的内容：

1. 会话摘要
   - 会话数量
   - 主要话题
   - 完成的任务

2. 用户偏好变化
   - 新提到的偏好
   - 习惯变化

3. 项目进展
   - 完成的任务
   - 遇到的问题
   - 待办事项

4. 学习新知
   - 技术栈信息
   - 团队信息
   - 项目背景
```

## 3.5.3 管理命令

### CLI 命令

```bash
# 查看今日记忆
openclaw daily

# 查看指定日期记忆
openclaw daily 2024-01-15
openclaw daily yesterday
openclaw daily last-week

# 添加今日记忆
openclaw daily add "用户提到喜欢使用 VSCode"

# 添加待办事项
openclaw daily todo "完成代码审查"

# 标记待办完成
openclaw daily done "完成代码审查"

# 查看所有待办
openclaw daily todos

# 搜索记忆
openclaw daily search "关键词"

# 列出最近记忆
openclaw daily list
```

### 手动创建

```bash
# 创建今日记忆文件
touch ~/.openclaw/workspace/daily/$(date +%Y-%m-%d).md

# 或使用 echo
echo "# $(date +%Y-%m-%d) 每日记忆" > ~/.openclaw/workspace/daily/$(date +%Y-%m-%d).md
```

## 3.5.5 配置选项

### 配置每日记忆

```json
// openclaw.json
{
  "workspace": {
    "memory": {
      "daily": {
        "enabled": true,
        "folder": "daily",
        "autoSave": true,
        "maxDays": 30,
        "includeSessionSummary": true,
        "includeUserPreferences": true,
        "includeTodos": true
      }
    }
  }
}
```

### 配置说明

| 选项 | 类型 | 默认值 | 说明 |
|-----|------|--------|------|
| enabled | boolean | true | 启用每日记忆 |
| folder | string | "daily" | 记忆文件夹 |
| autoSave | boolean | true | 自动保存 |
| maxDays | number | 30 | 保留天数 |
| includeSessionSummary | boolean | true | 包含会话摘要 |
| includeUserPreferences | boolean | true | 包含用户偏好 |
| includeTodos | boolean | true | 包含待办事项 |

## 3.5.6 最佳实践

### 内容建议

```
✅ 推荐写入每日记忆的内容：
- 今日完成的任务
- 重要讨论事项
- 用户偏好变化
- 项目进展
- 待办事项
- 学习到的新信息

❌ 不推荐写入的内容：
- 敏感信息（用 credentials）
- 长期知识（用 MEMORY.md）
- 大段代码（用项目文件）
- 调试日志
```

### 日常使用

```bash
# 1. 开始新的一天
openclaw daily
# 查看今天的记忆，了解昨天的工作

# 2. 添加待办
openclaw daily todo "完成代码审查"

# 3. 记录重要信息
openclaw daily add "用户提到下周一要演示"

# 4. 查看待办
openclaw daily todos

# 5. 标记完成
openclaw daily done "完成代码审查"
```

### 定期整理

```bash
# 每周整理
# 1. 查看本周记忆
openclaw daily list --week

# 2. 归档重要内容到 MEMORY.md
# 将重要的长期信息移到长期记忆

# 3. 清理过期记忆
# 自动清理超过 maxDays 的记忆
```

## 3.5.7 常见问题

### Q1: 每日记忆没有创建

```bash
# 检查配置
openclaw status | grep daily

# 检查文件夹权限
ls -la ~/.openclaw/workspace/daily/

# 手动创建
mkdir -p ~/.openclaw/workspace/daily
touch ~/.openclaw/workspace/daily/$(date +%Y-%m-%d).md
```

### Q2: 记忆没有自动保存

```bash
# 检查 autoSave 配置
# 确保 openclaw.json 中 "autoSave": true

# 检查磁盘空间
df -h ~/.openclaw/

# 手动保存
openclaw daily save
```

### Q3: 记忆文件太多

```bash
# 查看文件数量
ls ~/.openclaw/workspace/daily/ | wc -l

# 清理过期文件（超过 30 天）
find ~/.openclaw/workspace/daily/ -name "*.md" -mtime +30 -delete

# 或使用配置自动清理
# "maxDays": 30
```

## 本节小结

1. **每日记忆**：存储在 `daily/YYYY-MM-DD.md`，按日期自动创建
2. **内容类型**：今日事项、学习新知、待办事项、会话摘要
3. **加载时机**：每次会话自动加载当日记忆
4. **管理命令**：daily add/todo/done/list
5. **最佳实践**：自动记录 + 手动补充 + 定期整理
