```
目录名 → 架构层次（功能域）   文件章节号 → 学习顺序
─────────────────────────────────────────────────────
03_channels/  → Layer1（用户层）     8.x  → 第8章
03_gateway/   → Layer2（网关层）     10.x → 第12章
05_minimal_loop/ → Layer3（Agent层） 3.x  → 第3章
05_multi_agent/ → Layer3（Agent层）  7.x  → 第7章
02_config_models/ → Layer4（模型层）   4.x  → 第4章
07_tools_skills/ → Layer5（工具层）   5.x  → 第5章
16_system_layer/ → Layer6（系统层）  16.x → 第16章
```

**按功能查找** → 看目录名（架构层次）
**按学习顺序** → 看文件章节号（`x.x`）
