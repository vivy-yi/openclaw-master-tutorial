## 架构层次说明

docs/ 目录按 **OpenClaw 六层架构** 组织：

| 架构层次 | 目录 | 说明 |
|---------|------|------|
| Layer 1 用户层 | `03_channels/` | 渠道（用户入口：Telegram/飞书/Discord 等）|
| Layer 2 Gateway 层 | `03_gateway/` | 网关、认证、限流、协议 |
| Layer 3 Agent 层 | `05_minimal_loop/` | Agent 执行循环 |
| Layer 3 Agent 层 | `05_multi_agent/` | 多 Agent 协作 |
| Layer 4 Model 层 | `02_config_models/` | 模型配置 |
| Layer 5 Tools 层 | `07_tools_skills/` | 工具、Skills |
| Layer 6 System 层 | `16_system_layer/` | 系统操作 |
