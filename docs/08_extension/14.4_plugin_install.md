## 12.4.1 插件系统概述

### 什么是插件

```
┌─────────────────────────────────────────────────────────────┐
│                      插件 vs Bundle                         │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  原生插件 (Native Plugin)                                   │
│  ├── 运行在 Gateway 进程内的 TypeScript 模块                  │
│  ├── 可挂载自定义 Tools                                      │
│  ├── 注册全新聊天渠道                                        │
│  └── 需要 openclaw.plugin.json 清单                          │
│                                                              │
│  Bundle 插件                                                │
│  ├── Codex/Claude/Cursor 兼容包                             │
│  ├── 映射为 Skills/Hooks/MCP 配置                           │
│  └── 无需编写代码                                            │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 插件类型

| 类型 | 说明 | 示例 |
|-----|------|------|
| **Channel** | 消息渠道 | feishu, discord, telegram |
| **Provider** | 模型提供商 | google, anthropic |
| **Memory** | 记忆存储 | memory-core, memory-lancedb |
| **Tool** | 工具扩展 | 各类工具插件 |
| **Auth** | 认证方式 | minimax-portal-auth |

## 12.4.2 安装方式

### 方式一：CLI 安装（推荐）

```bash
# 列出可用插件
openclaw plugins list

# 从 npm 安装
openclaw plugins install @openclaw/feishu
openclaw plugins install @openclaw/discord
openclaw plugins install @openclaw/telegram

# 指定版本
openclaw plugins install @openclaw/feishu@2026.1.29

# 从本地路径安装
openclaw plugins install --path /path/to/plugin

# 从归档安装
openclaw plugins install --source archive /path/to/archive.tar.gz
```

### 方式二：手动安装

```bash
# 创建插件目录
mkdir -p ~/.openclaw/extensions

# 克隆插件
git clone https://github.com/openclaw/openclaw-extensions.git ~/.openclaw/extensions/feishu

# 或使用 npm
cd ~/.openclaw/extensions
npm install @openclaw/feishu
```

### 方式三：Docker 环境安装

```bash
# 在 Docker 中安装插件
docker exec openclaw openclaw plugins install @openclaw/feishu

# 或挂载插件目录
# docker-compose.yml
volumes:
  - ./plugins:/root/.openclaw/extensions
```

## 12.4.3 Bundle 安装

### 支持的 Bundle 格式

| 格式 | 来源 | 说明 |
|-----|------|------|
| **Codex** | .codex-plugin/ | Codex 插件包 |
| **Claude** | .claude-plugin/ | Claude 插件包 |
| **Cursor** | .cursor-plugin/ | Cursor 插件包 |

### 安装 Bundle

```bash
# 安装 Codex Bundle
openclaw plugins install ./my-codex-bundle

# 安装 Claude Bundle
openclaw plugins install ./my-claude-bundle

# 安装 Cursor Bundle
openclaw plugins install ./my-cursor-bundle

# 从归档安装
openclaw plugins install ./my-bundle.tgz
```

### Bundle 功能映射

| Bundle 功能 | OpenClaw 映射 |
|------------|--------------|
| `commands/` | Skills |
| `skills/` | Skills |
| `hooks/` (Codex 格式) | Hook Packs |
| `settings.json` | 嵌入式 Pi 设置 |
| `.mcp.json` | MCP 服务器配置 |

### 查看 Bundle 信息

```bash
# 查看插件详情
openclaw plugins info <bundle-id>

# 查看支持的特性
# 输出示例：
# - Format: bundle (codex)
# - Skills: commands/, skills/
# - Hooks: HOOK.md, handler.ts
# - MCP: 已配置
```

## 12.4.4 插件配置

### 启用插件

```json
{
  "plugins": {
    "enabled": true,
    "entries": {
      "feishu": {
        "enabled": true,
        "config": {
          "appId": "your-app-id",
          "appSecret": "your-app-secret",
          "domain": "feishu"
        }
      },
      "discord": {
        "enabled": true,
        "config": {
          "botToken": "your-bot-token"
        }
      }
    }
  }
}
```

### 使用 Config 命令配置

```bash
# 配置飞书
openclaw config set plugins.entries.feishu.config.appId "cli.a"
openclaw config set plugins.entries.feishu.config.appSecret "your-secret"
openclaw config set plugins.entries.feishu.config.domain "feishu"

# 配置 Telegram
openclaw config set plugins.entries.telegram.config.botToken "123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11"

# 配置 Discord
openclaw config set plugins.entries.discord.config.botToken "your-bot-token"
```

## 12.4.5 插件安全配置

### 白名单模式

```bash
# 只允许特定插件加载
openclaw config set plugins.allow '["feishu","telegram","discord"]'
```

### 黑名单模式

```bash
# 禁止特定插件
openclaw config set plugins.deny '["untrusted-plugin"]'
```

### 完整安全配置

```json
{
  "plugins": {
    "enabled": true,
    "allow": ["feishu", "telegram", "discord", "memory-core"],
    "deny": [],
    "load": {
      "paths": ["~/custom-plugins"]
    },
    "entries": {
      "feishu": { "enabled": true },
      "telegram": { "enabled": true }
    }
  }
}
```

## 12.4.6 插件管理命令

### 常用命令

```bash
# 列出所有已发现插件
openclaw plugins list

# 查看插件详情
openclaw plugins info <plugin-id>

# 启用插件
openclaw plugins enable <plugin-id>

# 禁用插件
openclaw plugins disable <plugin-id>

# 更新插件
openclaw plugins update
openclaw plugins update <plugin-id>

# 诊断插件问题
openclaw plugins doctor

# 卸载插件
openclaw plugins uninstall <plugin-id>
```

### 查看插件状态

```bash
# 基础状态
openclaw status

# 深度状态
openclaw status --deep

# 查看已加载的工具
openclaw tools list
```

## 12.4.7 插件目录结构

### 全局插件目录

```
~/.openclaw/extensions/
├── feishu/
│   ├── index.ts          # 入口文件
│   ├── package.json      # 依赖
│   ├── openclaw.plugin.json  # 清单
│   └── locales/          # 国际化
├── discord/
│   └── ...
└── telegram/
    └── ...
```

### 插件清单文件

每个原生插件必须包含 `openclaw.plugin.json`：

```json
{
  "id": "my-plugin",
  "name": "我的插件",
  "description": "插件描述",
  "version": "1.0.0",
  "configSchema": {
    "type": "object",
    "properties": {
      "apiKey": {
        "type": "string",
        "description": "API 密钥"
      }
    },
    "required": ["apiKey"]
  }
}
```

## 12.4.8 常见问题

### 问题一：插件未发现

```bash
# 检查插件目录
ls -la ~/.openclaw/extensions/

# 手动指定路径
openclaw config set plugins.load.paths '["/path/to/plugins"]'

# 重启 Gateway
openclaw gateway restart
```

### 问题二：插件加载失败

```bash
# 运行诊断
openclaw plugins doctor

# 查看详细日志
openclaw logs | grep -i plugin
```

### 问题三：配置验证失败

```bash
# 检查配置语法
openclaw config get plugins.entries

# 查看清单定义
openclaw plugins info <plugin-id>
```

### 问题四：Bundle 功能不生效

```bash
# 检查 Bundle 支持的功能
openclaw plugins info my-bundle

# 确保文件在正确目录
# Codex: .codex-plugin/
# Claude: .claude-plugin/
# Cursor: .cursor-plugin/
```

## 12.4.9 Docker 环境特殊配置

### 预装插件

```bash
# 构建时预装插件
export OPENCLAW_EXTENSIONS="feishu discord telegram"
./docker-setup.sh
```

### 运行时挂载

```yaml
# docker-compose.yml
services:
  openclaw:
    volumes:
      - ./plugins:/root/.openclaw/extensions:ro
```
