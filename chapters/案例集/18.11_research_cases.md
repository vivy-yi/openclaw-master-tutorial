## 14.11.1 AI财报追踪

### 配置



![CoT推理](./.assets/diagrams/130_chain_thought.png)

```bash
openclaw cron add \
  --name "Earnings Tracker" \
  --cron "0 21 * * *" \
  --message "追踪财报：
1. 检查即将发布的财报
2. 预分析预期
3. 发布后生成摘要
4. 关键指标对比" \
  --announce
```

## 14.11.3 预构建创意验证

### 场景描述

构建前自动扫描GitHub、HN、npm等，避免重复造轮子。

```bash
openclaw cron add \
  --name "Idea Validator" \
  --message "验证创意：
1. 搜索GitHub类似项目
2. 扫描HN讨论
3. 检查npm包
4. 评估市场竞争

如果市场拥挤建议放弃，否则继续"
```
