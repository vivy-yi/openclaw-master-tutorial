1. 浏览器控制截图（内置） ​

OpenClaw 内置 browser 工具，支持多种截图模式。

1.1 基本用法 ​

python
# 全页截图
browser(action=screenshot, url="https://openclaw.ai")

# 指定区域截图
browser(action=screenshot, selector="#main-content")

# 带延迟截图（等待 JS 加载）
browser(action=screenshot, url="https://openclaw.ai", delayMs=3000)

1.2 高级选项 ​

python
# 滚动截图（长页面）
browser(action=screenshot, url="https://long-page.com", fullPage=true)

# 元素截图
browser(action=screenshot, element=".modal-content")

# 自定义尺寸
browser(action=screenshot, url="https://openclaw.ai", width=1920, height=1080)

1.3 CDP 协议（无头浏览器） ​

python
# 使用 CDP 进行更高级的控制
browser(
    action=screenshot,
    url="https://openclaw.ai",
    CDP=true,
    options={
        "format": "png",
        "quality": 100,
        "captureBeyondViewport": true
    }
)

4.2 Puppeteer/Playwright（复杂交互） ​

对于需要先交互再截图的场景：

javascript
// puppeteer-screenshot.js
const puppeteer = require('puppeteer');

async function screenshot() {
    const browser = await puppeteer.launch();
    const page = await browser.newPage();
    
    // 登录后的页面
    await page.goto('https://app.openclaw.ai/dashboard');
    await page.type('#username', 'user');
    await page.type('#password', 'pass');
    await page.click('#login-btn');
    await page.waitForNavigation();
    
    // 截图
    await page.screenshot({ path: 'dashboard.png', fullPage: true });
    
    await browser.close();
}

screenshot();

8. 常见问题 ​

Q1: 截图模糊怎么办？ ​

解决方案：

  • 使用 deviceScaleFactor: 2 提高分辨率
  • 优先使用 PNG 而非 JPEG

Q2: 动态内容截图不完整？ ​

解决方案：

  • 增加 delayMs 等待加载
  • 使用 waitForSelector 等待特定元素

Q3: 需要登录后的截图？ ​

解决方案：

  • 使用 Playwright 配合 cookie
  • 或通过 API 获取登录状态
