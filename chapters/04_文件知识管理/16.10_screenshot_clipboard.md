## 10.2 screencapture - 屏幕截图

### 功能说明

截取屏幕或窗口截图。

### 参数

```json5
{
  "type": "screen",        // screen | window | region
  "display": 1,            // 显示器编号
  "region": {              // 区域截图
    "x": 0,
    "y": 0,
    "width": 1920,
    "height": 1080
  },
  "savePath": "/path/to/screenshot.png"  // 保存路径
}
```

### 使用示例

```bash
# 截取主屏幕
screencapture()

# 截取指定显示器
screencapture({ display: 2 })

# 截取窗口
screencapture({ type: "window", title: "Safari" })

# 截取指定区域
screencapture({
  type: "region",
  region: { x: 100, y: 100, width: 800, height: 600 }
})

# 保存到指定路径
screencapture({ savePath: "~/Desktop/screenshot.png" })
```

### 截图类型

| 类型 | 说明 |
|------|------|
| screen | 全屏截图 |
| window | 窗口截图 |
| region | 区域截图 |

## 10.3 clipboard_read - 读取剪贴板

### 功能说明

读取系统剪贴板内容。

### 参数

```json5
{
  "type": "text"  // text | image | files
}
```

### 使用示例

```bash
# 读取文本
clipboard_read()

# 读取文本（显式）
clipboard_read({ type: "text" })

# 读取图片
clipboard_read({ type: "image" })

# 读取文件列表
clipboard_read({ type: "files" })
```

---

## 10.4 clipboard_write - 写入剪贴板

### 功能说明

写入内容到系统剪贴板。

### 参数

```json5
{
  "text": "文本内容",      // 文本内容
  "image": "base64...",  // Base64 图片
  "files": ["path1"]     // 文件路径
}
```

### 使用示例

```bash
# 写入文本
clipboard_write({ text: "Hello World" })

# 复制代码
clipboard_write({ text: "const foo = 'bar';" })

# 复制文件路径
clipboard_write({ files: ["/path/to/file.pdf"] })
```

## 10.5 工具配置

### 全局配置

```json5
{
  tools: {
    allow: ["screencapture", "clipboard_read", "clipboard_write"]
  }
}
```

### 按 Agent 配置

```json5
{
  agents: {
    list: [
      {
        id: "assistant",
        tools: {
          allow: ["clipboard_read", "clipboard_write"],
          deny: ["screencapture"]  // 禁止截图
        }
      }
    ]
  }
}
```

## 10.6 实际应用场景

### 截图 + AI 分析

```bash
# 截屏并分析
screenshot = screencapture()
# 将截图发送给 AI 分析
```

### 复制粘贴自动化

```bash
# 读取当前选中的内容
content = clipboard_read()
# 处理内容
processed = process(content)
# 写回剪贴板
clipboard_write({ text: processed })
```
