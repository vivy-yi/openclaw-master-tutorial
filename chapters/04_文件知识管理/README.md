# 第4章 文件与知识管理

> **本章学习目标**: 掌握智能文件搜索和知识库构建
> **预计用时**: 60-90分钟
> **前置要求**: 完成基础部署

---

## 4.1 智能文件搜索

### 基本搜索

```
帮我找一下"发票"相关的文件
```

### 高级搜索

- 按类型搜索：`找所有PDF文件`
- 按日期搜索：`找上周修改的文件`
- 按内容搜索：`包含"项目计划"的文件`

---

## 4.2 构建个人知识库

### 知识库配置

```json
{
  "knowledge": {
    "enabled": true,
    "paths": [
      "~/Documents/notes",
      "~/Documents/wiki"
    ],
    "exclude": ["node_modules", ".git"]
  }
}
```

### 自动索引

OpenClaw会自动索引知识库内容，支持语义搜索。

---

## 4.3 批量文件处理

### 常见操作

```
# 合并PDF
合并 ~/Downloads/*.pdf

# 整理文件
按类型整理 ~/Desktop/*

# 批量重命名
重命名 ~/Photos/*.jpg --prefix "vacation_"
```

1. 智能搜索：支持类型、日期、内容搜索
2. 知识库：自动索引、语义搜索
