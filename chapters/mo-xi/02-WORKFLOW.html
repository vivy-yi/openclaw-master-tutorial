二、CDP 探索流程 ​

2.1 页面打开 ​

yaml
browser.open(url):
  - 输入: 任意 URL
  - 输出: 页面加载完成
  - 选项: profile="user" 维持登录态

2.2 结构分析 ​

yaml
browser.snapshot():
  - 获取页面 DOM 结构
  - 识别可交互元素
  - 发现数据容器
  - 输出: 结构化页面描述

2.3 网络拦截 ​

yaml
browser.intercept():
  - 拦截 XHR/Fetch 请求
  - 捕获 API 响应
  - 发现隐藏接口
  - 提取 JSON 数据

2.4 数据提取 ​

yaml
browser.evaluate(js):
  - 执行任意 JavaScript
  - 提取动态内容
  - 操作页面元素
  - 触发加载/点击

三、降级策略详解 ​

策略选择逻辑 ​

请求采集
    ↓
┌─────────────────┐
│ 尝试 API 直连   │ ← 最优先，效率最高
└─────────────────┘
    ↓ 失败/超时
┌─────────────────┐
│ CDP DOM 提取    │ ← 兜底方案，适用大多数
└─────────────────┘
    ↓ 失败/复杂
┌─────────────────┐
│ Screenshot + OCR │ ← 最后兜底，效率最低
└─────────────────┘

API 直连 ​

适用场景：

  • 公开数据接口
  • 已知 API 端点
  • 高效批量采集

示例 - 微博热搜：

javascript
// 拦截发现的 API
fetch('https://weibo.com/ajax/side/hotSearch')
  .then(r => r.json())
  .then(data => {
    data.data.realtime.forEach(item => {
      console.log(item.rank, item.word, item.num);
    });
  });

CDP DOM 提取 ​

适用场景：

  • 无公开 API
  • 需要登录态
  • 动态渲染内容

示例 - 微博用户信息：

javascript
// 提取页面元素数据
const users = document.querySelectorAll('.user-card');
users.forEach(user => {
  console.log({
    name: user.querySelector('.name')?.textContent,
    followers: user.querySelector('.followers')?.textContent
  });
});

Screenshot + OCR ​

适用场景：

  • 反爬防护
  • 复杂验证码
  • 图片数据

示例：

yaml
browser.screenshot():
  → 保存截图
  → OCR 识别文字
  → 解析结构化数据

四、自我学习机制 ​

学习时机 ​

时机学习内容
首次采集探索并记录方法
成功采集更新成功路径
失败采集记录失败原因
性能下降优化采集策略

学习内容 ​

json
{
  "platform": "平台名",
  "url_pattern": "URL 模式",
  "method": "采集方法",
  "endpoint": "API 端点",
  "selector": "DOM 选择器",
  "auth_required": "是否需登录",
  "performance": {
    "avg_duration": "平均耗时",
    "success_rate": "成功率",
    "last_success": "最后成功时间"
  }
}

知识复用 ​

新请求 → 知识库匹配
            ↓
      找到记录 → 直接使用记录的方法
            ↓
      未找到 → 重新探索 → 学习记录

六、错误处理 ​

重试机制 ​

yaml
retry:
  max_attempts: 3
  backoff: exponential
  delays: [1s, 5s, 30s]

降级:
  1. API → CDP DOM
  2. CDP DOM → Screenshot
  3. 全部失败 → 记录错误 + 通知用户

错误类型 ​

错误类型处理方式
网络超时重试 + 降级
登录失效提示用户重新登录
反爬拦截降低频率 + 切换 IP
页面加载失败重试 + 简化采集
