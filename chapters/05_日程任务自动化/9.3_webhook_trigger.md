## 9.3.1 Webhook 触发概述

### 触发原理

```
Webhook 触发 = 外部事件 → 自动执行

流程：
外部系统 → Webhook → 触发任务 → 执行动作

示例：
GitHub Push → OpenClaw → 代码审查
```

## 9.3.2 配置 Webhook

### 触发器配置

```json
{
  "webhook_triggers": {
    "enabled": true,
    "triggers": [
      {
        "id": "github_push",
        "name": "GitHub 代码推送",
        "event": "push",
        "source": "github",
        "filter": {
          "branches": ["main", "develop"]
        },
        "action": {
          "type": "workflow",
          "workflow": "code_review"
        }
      }
    ]
  }
}
```

### 事件类型

```
支持的 Webhook 事件：

GitHub/GitLab：
- push: 代码推送
- pull_request: PR 事件
- issues: Issue 事件
- release: 发布事件

通用：
- http: HTTP 请求
- schedule: 定时触发
- manual: 手动触发
```

## 9.3.3 过滤条件

### 分支过滤

```json
{
  "filter": {
    "branches": ["main", "develop", "feature/*"],
    "exclude_branches": ["test-*"]
  }
}
```

### 路径过滤

```json
{
  "filter": {
    "paths": ["src/**", "*.go"],
    "exclude_paths": ["*.test.go", "docs/**"]
  }
}
```

### 内容过滤

```json
{
  "filter": {
    "commits": {
      "contains": ["fix", "bug"],
      "author": ["@username"]
    }
  }
}
```

## 9.3.4 处理数据

### 数据映射

```json
{
  "mapping": {
    "source": "github",
    "target": {
      "branch": "{{ref}}",
      "commit": "{{after}}",
      "author": "{{pusher.name}}",
      "files": "{{modified}}"
    }
  }
}
```

### 数据转换

```python
# 自定义数据处理
def transform_github_event(event):
    return {
        'branch': event['ref'].replace('refs/heads/', ''),
        'commit': event['after'],
        'changes': analyze_changes(event['commits'])
    }
```

## 9.3.5 响应处理

### 响应配置

```json
{
  "response": {
    "on_success": {
      "comment": {
        "platform": "github",
        "body": "代码审查完成\n\n{{review_summary}}"
      }
    },
    "on_failure": {
      "notify": {
        "channels": ["slack"],
        "message": "自动化任务失败: {{error}}"
      }
    }
  }
}
```

## 9.3.6 安全配置

### 签名验证

```json
{
  "security": {
    "signature": {
      "enabled": true,
      "header": "X-Hub-Signature-256",
      "secret": "{{github_webhook_secret}}"
    }
  }
}
```

### IP 白名单

```json
{
  "security": {
    "ip_whitelist": [
      "140.82.112.0/20"  // GitHub
    ]
  }
}
```
