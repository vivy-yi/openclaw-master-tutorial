## 立即开始：项目1 - 客服机器人

### 第一步：创建基础架构

```yaml
# 客服机器人功能
- 自动回复常见问题
- 收集用户信息
- 转接人工
- 记录对话历史
```

### 第二步：配置渠道

```json
{
  "channels": {
    "telegram": { "enabled": true },
    "feishu": { "enabled": true }
  }
}
```

### 第三步：定义 FAQ

```
常见问题库：
1. 你们提供什么服务？
2. 价格是多少？
3. 如何联系你们？
4. 付款方式有哪些？
```

### 第四步：设置自动回复规则

```
关键词匹配 -> 自动回复
未匹配 -> 转人工
```
