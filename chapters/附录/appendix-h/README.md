## v3.0.0 (2025-03-15)

### 🎉 重大更新
- **多Agent架构**：支持Subagent协调，复杂任务自动分解
- **MCP协议2.0**：更强大的工具集成能力
- **云端同步**：配置和对话跨设备同步

### ✨ 新特性
- 新增可视化任务监控面板
- 支持自定义Agent角色和人格
- 集成20+新的MCP Server
- 增强的代码补全功能

### 🔧 改进
- 启动速度提升40%
- 内存占用降低30%
- 中文处理能力优化

### ⚠️ 破坏性变更
- `claude` 配置项更名为 `agents`
- MCP Server配置格式变更
- 旧版Skill需要更新manifest格式

**迁移指南**：
```bash
# 自动迁移
openclaw migrate --from 2.x --to 3.0

# 手动检查
openclaw doctor
```

## 升级指南

### 自动升级

```bash
# 检查更新
openclaw version --check

# 执行升级
curl -fsSL https://openclaw.ai/install.sh | bash

# 验证版本
openclaw version
```

### 手动升级

```bash
# 1. 备份配置
cp -r ~/.openclaw ~/.openclaw.backup

# 2. 下载新版本
# 根据系统选择对应包

# 3. 安装
# macOS/Linux
sudo dpkg -i openclaw_*.deb
# 或
sudo rpm -i openclaw_*.rpm

# 4. 重启服务
openclaw restart
```

### 版本锁定

```json
{
  "version": {
    "lock": true,
    "channel": "stable"
  }
}
```
