## 12.5.2 飞书插件配置

### 前提条件

1. 飞书开放平台账号
2. 创建企业自建应用
3. 获取 App ID 和 App Secret

### 创建飞书应用

1. 访问 [飞书开放平台](https://open.feishu.cn/)
2. 创建企业自建应用
3. 添加权限：
   - `im:chat:ReadOnly`
   - `im:chat:SendAsBot`
   - `im:message:SendAsBot`

### 获取配置信息

```
App ID: cli_xxxxxxxxxxxxxx
App Secret: your-app-secret
```

### 配置 OpenClaw

```bash
# 安装插件
openclaw plugins install @openclaw/feishu

# 配置
openclaw config set plugins.entries.feishu.enabled true
openclaw config set plugins.entries.feishu.config.appId "cli_xxxxxxxxxxxxxx"
openclaw config set plugins.entries.feishu.config.appSecret "your-app-secret"
openclaw config set plugins.entries.feishu.config.domain "feishu"
```

### 配置 JSON

```json
{
  "plugins": {
    "entries": {
      "feishu": {
        "enabled": true,
        "config": {
          "appId": "cli_xxxxxxxxxxxxxx",
          "appSecret": "your-app-secret",
          "domain": "feishu"
        }
      }
    }
  }
}
```

### 绑定消息

```json
{
  "bindings": [
    {
      "agentId": "main",
      "match": {
        "channel": "feishu"
      }
    }
  ]
}
```

## 12.5.3 Discord 插件配置

### 前提条件

1. Discord 账号
2. 创建 Discord 应用
3. 获取 Bot Token

### 创建 Discord Bot

1. 访问 [Discord Developer Portal](https://discord.com/developers/applications)
2. 创建新应用
3. 添加 Bot
4. 获取 Token
5. 邀请 Bot 到服务器

### 所需权限

```
- Read Messages/View Channels
- Send Messages
- Manage Messages
- Embed Links
- Read Message History
- Use Slash Commands
```

### 配置 OpenClaw

```bash
# 安装插件
openclaw plugins install @openclaw/discord

# 配置
openclaw config set plugins.entries.discord.enabled true
openclaw config set plugins.entries.discord.config.botToken "your-bot-token"
```

### 配置 JSON

```json
{
  "plugins": {
    "entries": {
      "discord": {
        "enabled": true,
        "config": {
          "botToken": "MTQ4..."
        }
      }
    }
  }
}
```

### 高级配置

```json
{
  "channels": {
    "discord": {
      "enabled": true,
      "groupPolicy": "allowlist",
      "guilds": {
        "123456789": {
          "channels": {
            "general": { "allow": true },
            "bot-commands": { "allow": true }
          },
          "requireMention": false
        }
      }
    }
  }
}
```

## 12.5.4 Telegram 插件配置

### 前提条件

1. Telegram 账号
2. 与 @BotFather 对话创建机器人
3. 获取 Bot Token

### 创建 Telegram Bot

1. 在 Telegram 搜索 @BotFather
2. 发送 `/newbot` 命令
3. 按照指示命名机器人
4. 获取 Token

### 配置 OpenClaw

```bash
# 安装插件
openclaw plugins install @openclaw/telegram

# 配置
```

### 配置 JSON

```json
{
  "plugins": {
    "entries": {
      "telegram": {
        "enabled": true,
        "config": {
          "botToken": "123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11"

      }
    }
  }
}
```

## 12.5.6 Slack 插件配置

### 前提条件

1. Slack Workspace
2. 创建 Slack App
3. 获取 Bot Token

### 创建 Slack App

1. 访问 [Slack API](https://api.slack.com/apps)
2. 创建新应用
3. 添加 Bot Token Scope
4. 安装到 Workspace

### 所需权限

```
- chat:write
- channels:read
- groups:read
- users:read
- message:read
```

### 配置 OpenClaw

```bash
# 安装插件
openclaw plugins install @openclaw/slack

# 配置
openclaw config set plugins.entries.slack.enabled true
openclaw config set plugins.entries.slack.config.botToken "xoxb-..."
```

### 配置 JSON

```json
{
  "plugins": {
    "entries": {
      "slack": {
        "enabled": true,
        "config": {
          "botToken": "xoxb-your-token",
          "signingSecret": "your-signing-secret"
        }
      }
    }
  }
}
```

## 12.5.7 记忆插件配置

### memory-core（内置）

记忆核心功能默认启用，无需额外配置。

```json
{
  "memory": {
    "enabled": true,
    "search": {
      "enabled": true
    }
  }
}
```

### memory-lancedb（向量记忆）

```bash
# 安装插件
openclaw plugins install @openclaw/memory-lancedb

# 配置
openclaw config set plugins.entries.memory-lancedb.enabled true
openclaw config set plugins.entries.memory-lancedb.config.persistDirectory "~/.openclaw/memory"
```

### 配置 JSON

```json
{
  "plugins": {
    "entries": {
      "memory-lancedb": {
        "enabled": true,
        "config": {
          "persistDirectory": "~/.openclaw/memory",
          "dimensions": 1536,
          "metric": "cosine"
        }
      }
    }
  }
}
```

### 使用记忆功能

```bash
# 搜索记忆
openclaw memory search "关于项目的细节"

# 列出记忆
openclaw memory list
```

```bash
# 配置环境变量
export ANTHROPIC_API_KEY="sk-ant-..."

```

```bash
# 配置

# 或使用配置
```

```bash
# 安装插件
openclaw plugins install @openclaw/google
```

## 12.5.9 MCP 插件配置

### 什么是 MCP

MCP (Model Context Protocol) 是用于连接 AI 模型和外部服务的协议。

### 安装 MCP 服务器

```bash
# 安装 MCP 插件
openclaw plugins install @openclaw/mcp

# 或从 npm 安装
npm install -g @modelcontextprotocol/server-filesystem
```

### 配置 MCP

```json
{
  "mcp": {
    "servers": {
      "filesystem": {
        "command": "npx",
        "args": ["-y", "@modelcontextprotocol/server-filesystem", "./workspace"]
      }
    }
  }
}
```

### 常用 MCP 服务器

| 服务器 | 功能 |
|-------|------|
| @modelcontextprotocol/server-filesystem | 文件系统访问 |
| @modelcontextprotocol/server-github | GitHub API |
| @modelcontextprotocol/server-brave-search | Brave 搜索 |
| @modelcontextprotocol/server-git | Git 操作 |

## 12.5.11 常见问题

### 渠道消息收不到

```bash
# 检查插件状态
openclaw plugins list

# 检查配置
openclaw config get plugins.entries

# 查看日志
openclaw logs | grep -i "feishu\|discord\|telegram"
```

### Bot 没有响应

```bash
# 检查事件订阅
# 飞书：检查事件订阅 URL
# Discord：检查 Intent 配置
# Telegram：检查 bot Token
```

### 记忆不工作

```bash
# 检查记忆插件
openclaw plugins info memory-lancedb

# 测试记忆搜索
openclaw memory search "test query"
```
