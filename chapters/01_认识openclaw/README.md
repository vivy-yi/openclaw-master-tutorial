## 1.1 背景与诞生

### 从"对话"到"执行"的AI进化

2022年，ChatGPT横空出世，让全世界看到了大语言模型的威力。但很快，人们发现了一些局限：

| 局限 | 具体表现 |
|-----|---------|
| **只能对话** | "帮我整理桌面文件" → "我可以告诉你如何整理..." |
| **无法访问本地** | 不能读取你的文件、日历、邮件 |
| **没有记忆** | 每次对话都是重新开始 |
| **数据上传** | 敏感文件需要上传到第三方服务器 |

OpenClaw的创造者Peter Steinberger也有同样的困扰。他想要一个能够**真正帮他做事**的AI助手。

2025年11月，他将这个工具开源，命名为**Clawdbot**。没想到，这个项目迅速引爆GitHub：
- 一周内获得100,000+ Stars
- 超过Linux成为GitHub历史上增长最快的开源项目

经过短暂更名（Moltbot）后，最终定名为**OpenClaw**。

## 1.3 工作原理

### 架构概览

```
用户层 → Gateway层 → Agent层 → 工具层 → 系统层
```

当用户在飞书发送消息时：
1. **用户发送**: "帮我整理下载文件夹"
2. **飞书Bot接收**: 转发到OpenClaw Gateway
3. **Gateway路由**: 确定Agent处理
4. **Agent处理**: LLM理解 → 任务规划 → 工具选择
5. **执行操作**: 调用read/exec工具
6. **返回结果**: 通过飞书Bot返回

### 核心配置文件

`openclaw.json` 位于 `~/.openclaw/`，定义：
- 使用的AI模型和API密钥
- 启用的聊天渠道
- 安装的技能
- 安全设置

## 1.4 成本分析

| 方案 | 月费用 | 适用场景 |
|-----|-------|---------|
| 本地部署 | 0元 | 有可用电脑 |
| 腾讯云轻量 | 20元 | 需要24/7在线 |
| 火山引擎 | 9.9元 | 性价比之选 |

API成本（必须）:
| 模型 | 月费用估算 | 特点 |
|-----|-----------|-----|
| DeepSeek | 5-15元 | 性价比高 |
| Kimi | 10-30元 | 长文档处理强 |
| Claude | 50-200元 | 能力强但贵 |

**总计**: 最低5元/月，对比ChatGPT Plus 140元/月可节省73%-96%

## 1.5 硬件与软件要求

**最低配置**:
- CPU: 任意现代处理器
- 内存: 2GB RAM
- 存储: 1GB可用空间

**推荐配置**:
- CPU: 4核+
- 内存: 4GB+ RAM
- 存储: 10GB+ SSD

**软件要求**:
- Node.js >= 22.0
- 操作系统: macOS / Linux / Windows (WSL)

## 常见问题

**Q1: OpenClaw和ChatGPT有什么区别？**
A: ChatGPT只能回答问题；OpenClaw可以执行实际任务。

**Q2: 我需要会编程才能用OpenClaw吗？**
A: 基础使用不需要编程。深度定制需要JavaScript/Node.js基础。

**Q3: OpenClaw安全吗？**
A: 相对安全。数据保存在自己设备上，但要注意安全策略配置。

**Q4: 没有服务器可以运行OpenClaw吗？**
A: 可以！可以在自己电脑运行。需要24/7在线建议用云服务器。
