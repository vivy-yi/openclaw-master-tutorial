## 7.2 process - 进程管理

### 功能说明

管理后台执行的进程，查看状态、获取输出、终止进程。

### 参数

```json5
{
  "action": "list",           // 操作类型
  "sessionId": "xxx"          // 进程会话 ID
}
```

### 操作类型

| 操作 | 说明 |
|------|------|
| list | 列出所有后台进程 |
| output | 获取进程输出 |
| kill | 终止进程 |
| wait | 等待进程完成 |

### 使用示例

#### 列出进程

```bash
# 列出所有后台进程
process("list")

# 获取特定进程输出
process("output", { sessionId: "a1b2c3d4" })

# 终止进程
process("kill", { sessionId: "a1b2c3d4" })

# 等待进程完成
process("wait", { sessionId: "a1b2c3d4", timeout: 60000 })
```

### 与 exec 配合使用

```bash
# 后台执行命令
exec("npm run dev", { background: true })
# 返回: { status: "running", sessionId: "xxx" }

# 查看进程状态
process("list")

# 获取输出
process("output", { sessionId: "xxx" })

# 终止进程
process("kill", { sessionId: "xxx" })
```

## 7.3 canvas - 画布操作

### 功能说明

实时画布操作，支持多人协作。

### 参数

```json5
{
  "action": "write",      // 操作类型
  "content": "内容",      // 写入内容
  "append": true          // 是否追加
}
```

### 操作类型

| 操作 | 说明 |
|------|------|
| write | 写入画布 |
| read | 读取画布 |
| clear | 清空画布 |
| list | 列出画布 |

### 使用示例

#### 写入画布

```bash
# 写入内容
canvas_write("Hello World")

# 追加内容
canvas_write("\nNew line", { append: true })

# 写入 JSON
canvas_write(JSON.stringify({ status: "ok" }))
```

#### 读取画布

```bash
# 读取整个画布
canvas_read()

# 读取最后 N 行
canvas_read({ tail: 100 })
```

#### 清空画布

```bash
# 清空画布内容
canvas_clear()
```

## 7.4 工具配置

### 全局配置

```json5
{
  tools: {
    allow: ["process", "canvas", "canvas_write", "canvas_read"]
  }
}
```

### 按 Agent 配置

```json5
{
  agents: {
    list: [
      {
        id: "worker",
        tools: {
          allow: ["process", "exec"]
        }
      },
      {
        id: "collaborator",
        tools: {
          allow: ["canvas", "canvas_write", "canvas_read"]
        }
      }
    ]
  }
}
```

## 7.5 实时协作配置

### 画布服务配置

```json5
{
  agents: {
    defaults: {
      canvas: {
        // 启用实时协作
        realtime: true,
        // 最大存储大小（MB）
        maxSizeMB: 10,
        // 自动保存间隔（秒）
        autoSaveInterval: 30
      }
    }
  }
}
```
