## 8.2 voice_call - 语音通话

### 功能说明

控制语音通话，包括发起、继续、结束和状态查询。

### 参数

```json5
{
  "action": "initiate_call",   // 操作类型
  "to": "+1234567890",        // 目标号码
  "message": "Hello",         // 接通后播放的消息
  "mode": "notify"            // 模式：notify(通知) 或 conversation(对话)
}
```

### 操作类型

| 操作 | 说明 |
|------|------|
| initiate_call | 发起新通话 |
| continue_call | 继续通话 |
| speak_to_user | 播报消息 |
| end_call | 结束通话 |
| get_status | 获取通话状态 |

### 使用示例

#### 发起通话

```bash
# 发起通知型通话
voice_call("initiate_call", {
  to: "+1234567890",
  message: "您的订单已发货",
  mode: "notify"
})

# 发起对话型通话
voice_call("initiate_call", {
  to: "+1234567890",
  message: "您好，请问有什么可以帮助您？",
  mode: "conversation"
})
```

#### 继续通话

```bash
# 继续通话并发送消息
voice_call("continue_call", {
  callId: "call-xxx",
  message: "请稍等，我为您查询订单状态"
})
```

#### 播报消息

```bash
# 播报消息（不改变通话状态）
voice_call("speak_to_user", {
  callId: "call-xxx",
  message: "您的订单正在处理中，预计今天下午送达"
})
```

#### 结束通话

```bash
# 结束通话
voice_call("end_call", {
  callId: "call-xxx"
})
```

#### 查询状态

```bash
# 获取通话状态
voice_call("get_status", {
  callId: "call-xxx"
})
# 返回: { status: "in-progress", duration: 120, ... }
```

## 8.3 语音配置

### 全局配置

```json5
{
  tools: {
    allow: ["group:voice"]
  }
}
```

### 按 Agent 配置

```json5
{
  agents: {
    list: [
      {
        id: "caller",
        tools: {
          allow: ["voice_call"],
          deny: ["voice_end"]  // 禁止结束通话
        }
      }
    ]
  }
}
```

### 通话服务配置

```json5
{
  agents: {
    defaults: {
      voice: {
        // 默认通话模式
        defaultMode: "notify",
        // 最大通话时长（秒）
        maxDuration: 300,
        // 允许的国际号码
        allowInternational: false,
        // 黑名单
        blockedNumbers: ["+1234567890"]
      }
    }
  }
}
```

## 8.4 平台原生工具

### 概述

各消息平台提供原生工具：

| 平台 | 工具 | 说明 |
|------|------|------|
| WhatsApp | whatsapp | 发送消息、图片、文件 |
| Telegram | telegram | 发送消息、媒体、按钮 |
| Discord | discord | 发送消息、Embeds |
| Slack | slack | 发送消息、块 |
| Zalo | zalo | 发送消息、图片、链接 |

### WhatsApp 示例

```bash
# 发送文本消息
whatsapp("send", {
  to: "+1234567890",
  message: "Hello!"
})

# 发送图片
whatsapp("image", {
  to: "+1234567890",
  url: "https://example.com/image.jpg",
  caption: "图片说明"
})

# 发送文件
whatsapp("file", {
  to: "+1234567890",
  url: "https://example.com/doc.pdf"
})
```

### Telegram 示例

```bash
# 发送消息
telegram("send", {
  chatId: "123456789",
  text: "Hello!"
})

# 发送带按钮的消息
telegram("send", {
  chatId: "123456789",
  text: "请选择:",
  replyMarkup: {
    inline_keyboard: [
      [{ text: "选项1", callback_data: "option1" }],
      [{ text: "选项2", callback_data: "option2" }]
    ]
  }
})
```
