## 8.4.2 应用创建

### 创建步骤

```
Discord 应用创建流程：

1. 访问 https://discord.com/developers/applications
2. 点击「New Application」
3. 填写应用名称
4. 获取 Application ID
5. 创建 Bot 用户
6. 获取 Bot Token
7. 配置权限
8. 邀请到服务器
```

### 权限配置

```
必需权限：

- SEND_MESSAGES
- EMBED_LINKS
- ATTACH_FILES
- USE_EXTERNAL_EMOJIS
- MANAGE_MESSAGES
- READ_MESSAGE_HISTORY
- ADD_REACTIONS
- USE_SLASH_COMMANDS
```

## 8.4.3 配置 OpenClaw

### 基础配置

```json
{
  "channels": {
    "discord": {
      "enabled": true,
      "config": {
        "application_id": "123456789012345678",
        "token": "your-bot-token",
        "intents": [
          "GUILDS",
          "GUILD_MESSAGES",
          "MESSAGE_CONTENT"
        ],
        "webhook_path": "/webhook/discord"
      }
    }
  }
}
```

### 邀请链接

```
邀请 Bot 到服务器：

URL 格式：
https://discord.com/api/oauth2/authorize?client_id=APP_ID&permissions=PERMISSIONS&scope=bot

示例：
https://discord.com/api/oauth2/authorize?client_id=123456789012345678&permissions=8&scope=bot
```

## 8.4.6 常见问题

### Q1: Bot 无法加入服务器

```
解决：
1. 检查权限
2. 确认邀请链接正确
3. 检查服务器人数限制
```

### Q2: 消息发送失败

```
解决：
1. 检查 Bot 是否在频道
2. 验证权限
3. 检查消息长度
```

### Q3: Intent 错误

```
错误：Intent is not enabled

解决：
1. 在开发者后台启用 Intent
2. 在配置中声明 Intent
3. 重新邀请 Bot
```
