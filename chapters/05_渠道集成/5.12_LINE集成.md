# LINE 集成

> 日本和台湾主流通讯工具的 OpenClaw 接入配置。

## 准备工作

1. 注册 [LINE Developers](https://developers.line.biz/) 账号
2. 创建一个 Messaging API 频道
3. 获取 Channel ID、Channel Secret 和 Access Token

## OpenClaw 配置

```json
{
  "channels": {
    "line": {
      "enabled": true,
      "channelId": "your-channel-id",
      "channelSecret": "your-channel-secret",
      "accessToken": "your-access-token"
    }
  }
}
```

## Webhook 配置

在 LINE Developers Console 中设置 Webhook URL：
```
https://your-gateway-domain.com/api/line/webhook
```

## 验证连接

```bash
# 重启 Gateway
openclaw gateway restart

# 测试 Webhook
curl -X POST https://your-gateway-domain.com/api/line/webhook \
  -H "Content-Type: application/json" \
  -d '{"events": []}'
```

## 特性说明

- LINE 支持 Rich Menu（富菜单）
- 支持文本、图片、贴纸等多种消息类型
- LINE 官方强调消息必须来自真实用户，不可批量发送推广信息
