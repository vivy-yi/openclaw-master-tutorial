# 5.6 内置 Skills 详解

## 本节目标
- 掌握 OpenClaw 内置 Skills 的完整目录
- 理解各 Skills 的功能与文件位置
- 学会按需查找和启用 Skills

---

## 5.6.1 内置 Skills 概览

### 什么是内置 Skills

```
┌─────────────────────────────────────────────────────────────┐
│                    内置 Skills                               │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  OpenClaw 开箱即用的 Skills：                              │
│  ├── 无需安装，直接使用                                     │
│  ├── 经过充分测试，稳定可靠                                 │
│  └── 持续维护更新                                         │
│                                                              │
│  Skills 数量：120+                                          │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### Skills 存放位置

| 来源 | 路径 | 说明 |
|------|------|------|
| npm 全局包 | `/opt/homebrew/lib/node_modules/openclaw/skills/` | OpenClaw 官方内置 |
| 用户目录 | `~/.openclaw/skills/` | 用户自定义或扩展 |
| Agent 共享 | `~/.agents/skills/` | 多 Agent 共享技能 |

## 5.6.2 一、飞书集成类（Feishu）

| 技能名称 | 文件位置 | 功能 |
|---------|---------|------|
| feishu-doc | openclaw 内置 | 飞书云文档读写（读/写/创建/插入表格） |
| feishu-drive | openclaw 内置 | 飞书云空间管理（列表/创建文件夹/移动/删除） |
| feishu-wiki | openclaw 内置 | 飞书知识库导航（空间/节点/搜索） |
| feishu-bitable | openclaw 内置 | 飞书多维表格全功能管理（增删改查字段/记录） |
| feishu-chat | openclaw 内置 | 飞书群聊操作（成员/信息） |
| feishu-app-scopes | openclaw 内置 | 飞书应用权限查询与调试 |
| feishu-bitable-get-meta | openclaw 内置 | 解析飞书多维表格 URL，获取 app_token/table_id |

**使用示例**：

```markdown
用户：将这份报告同步到飞书知识库
→ 使用 feishu-wiki 创建节点
→ 使用 feishu-doc 写入内容
```

## 5.6.12 调用内置 Skills

### 自动调用（推荐）

```
用户：帮我查询一下明天上海的天气
→ LLM 自动识别需要 weather skill
← 返回天气预报
```

### 手动指定

```
用户：使用 github skill 查看最近 5 个 issues
→ 明确指定调用 github skill
← 返回 Issues 列表
```

### 查看所有可用 Skills

```bash
# 通过 clawhub 查看
clawhub search <关键词>

# 通过 OpenClaw CLI
openclaw skills list

# 查看 skills 状态
openclaw status
```

## 5.6.13 Skills 配置

### 启用/禁用

```json
{
  "skills": {
    "enabled": true,
    "autoLoad": true
  }
}
```

### 自定义 Skills 路径

```json
{
  "skills": {
    "paths": [
      "~/.openclaw/skills",
      "./custom-skills"
    ]
  }
}
```
