5.6 内置 Skills 详解 ​

本节目标 ​

  • 掌握 OpenClaw 内置 Skills 的完整目录
  • 理解各 Skills 的功能与文件位置
  • 学会按需查找和启用 Skills

5.6.1 内置 Skills 概览 ​

什么是内置 Skills ​

┌─────────────────────────────────────────────────────────────┐
│                    内置 Skills                               │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  OpenClaw 开箱即用的 Skills：                              │
│  ├── 无需安装，直接使用                                     │
│  ├── 经过充分测试，稳定可靠                                 │
│  └── 持续维护更新                                         │
│                                                              │
│  Skills 数量：120+                                          │
│                                                              │
└─────────────────────────────────────────────────────────────┘

Skills 存放位置 ​

来源路径说明
npm 全局包/opt/homebrew/lib/node_modules/openclaw/skills/OpenClaw 官方内置
用户目录~/.openclaw/skills/用户自定义或扩展
Agent 共享~/.agents/skills/多 Agent 共享技能

5.6.2 一、飞书集成类（Feishu） ​

技能名称文件位置功能
feishu-docopenclaw 内置飞书云文档读写（读/写/创建/插入表格）
feishu-driveopenclaw 内置飞书云空间管理（列表/创建文件夹/移动/删除）
feishu-wikiopenclaw 内置飞书知识库导航（空间/节点/搜索）
feishu-bitableopenclaw 内置飞书多维表格全功能管理（增删改查字段/记录）
feishu-chatopenclaw 内置飞书群聊操作（成员/信息）
feishu-app-scopesopenclaw 内置飞书应用权限查询与调试
feishu-bitable-get-metaopenclaw 内置解析飞书多维表格 URL，获取 app_token/table_id

使用示例：

markdown
用户：将这份报告同步到飞书知识库
→ 使用 feishu-wiki 创建节点
→ 使用 feishu-doc 写入内容

markdown

quantitative-factors量化因子分析（多因子模型/统计套利/机器学习/回测）

alternative-investment另类资产分析（PE/大宗商品/对冲基金/数字资产）

使用示例：

markdown
用户：帮我分析一下贵州茅台的投资价值
→ 使用 seven-dimension-analysis skill
→ 并行调用：技术面/基本面/宏观数据/情绪面
← 返回完整七维度分析报告

用户：设置茅台价格跌破 1500 元的提醒
→ 使用 prediction-tracker 记录预测
→ 使用 cron 设置价格监控

5.6.12 调用内置 Skills ​

自动调用（推荐） ​

用户：帮我查询一下明天上海的天气
→ LLM 自动识别需要 weather skill
← 返回天气预报

手动指定 ​

用户：使用 github skill 查看最近 5 个 issues
→ 明确指定调用 github skill
← 返回 Issues 列表

查看所有可用 Skills ​

bash
# 通过 clawhub 查看
clawhub search <关键词>

# 通过 OpenClaw CLI
openclaw skills list

# 查看 skills 状态
openclaw status

5.6.13 Skills 配置 ​

启用/禁用 ​

json
{
  "skills": {
    "enabled": true,
    "autoLoad": true
  }
}

自定义 Skills 路径 ​

json
{
  "skills": {
    "paths": [
      "~/.openclaw/skills",
      "./custom-skills"
    ]
  }
}
