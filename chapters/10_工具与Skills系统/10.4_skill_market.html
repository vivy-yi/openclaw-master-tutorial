5.4.3 技能搜索 ​

CLI 搜索 ​

bash
# 搜索技能
openclaw skills search code-review

# 列出所有技能
openclaw skills list

# 查看技能详情
openclaw skills info code-review

# 查看分类
openclaw skills categories

关键词搜索 ​

搜索技巧：

1. 精确匹配
   code-review

2. 关键词组合
   api design

3. 分类筛选
   category:frontend

4. 评分筛选
   rating:>4.5

筛选条件 ​

筛选选项：
├── 分类：category
├── 评分：rating
├── 下载量：downloads
├── 更新：updated
├── 免费/付费：price
└── 语言：language

5.4.4 技能安装 ​

安装命令 ​

bash
# 安装技能
openclaw skills install code-review

# 指定版本
openclaw skills install code-review@1.2.0

# 从 URL 安装
openclaw skills install https://github.com/xxx/skill

# 批量安装
openclaw skills install -f skills.txt

安装流程 ​

安装步骤：

1. 下载技能包
   → 获取 SKILL.md、工具配置等

2. 验证完整性
   → 检查必要文件
   → 验证签名

3. 安装到本地
   → 复制到 skills 目录
   → 注册到系统

4. 测试运行
   → 执行示例命令
   → 验证功能

本地管理 ​

本地技能目录：
~/.openclaw/skills/
├── code-review/
├── unit-test-gen/
└── custom-skill/

管理命令：
├── 列出本地：openclaw skills local
├── 更新：openclaw skills update
├── 删除：openclaw skills remove
└── 导出：openclaw skills export

5.4.5 技能使用 ​

在对话中使用 ​

使用方式：

用户：@code-review /path/to/file

或

用户：使用代码审查技能
→ 自动匹配最佳 Skill
← 返回审查结果

配置默认技能 ​

json
{
  "agent": {
    "default_skills": [
      "code-review",
      "git-helper"
    ]
  }
}

技能组合 ​

json
{
  "agent": {
    "workflow": [
      "code-review",
      "unit-test-gen",
      "docs-gen"
    ]
  }
}

5.4.6 技能评分与评价 ​

评分系统 ​

评分维度：
├── 功能完整性：功能是否齐全
├── 使用体验：操作是否流畅
├── 文档质量：说明是否清晰
├── 维护活跃度：更新是否及时
└── 社区支持：问题响应速度

评分等级：
⭐⭐⭐⭐⭐ 优秀
⭐⭐⭐⭐ 良好
⭐⭐⭐ 一般
⭐⭐ 待改进
⭐ 差

评价方法 ​

用户可以：
1. 评分（1-5 星）
2. 评价（文字反馈）
3. 报告问题
4. 提出建议

评价内容：
- 功能是否符合描述
- 使用中遇到的问题
- 改进建议

5.4.7 常见问题 ​

Q1: 技能安装失败 ​

错误：Install failed

解决：
1. 检查网络连接
2. 确认技能存在
3. 验证权限
4. 查看错误日志

Q2: 技能不兼容 ​

错误：Incompatible skill version

解决：
1. 检查 OpenClaw 版本
2. 安装兼容版本
3. 更新 OpenClaw

Q3: 技能加载慢 ​

问题：技能加载慢

解决：
1. 使用本地缓存
2. 预加载常用技能
3. 优化网络
