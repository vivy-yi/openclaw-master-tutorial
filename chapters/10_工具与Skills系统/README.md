# 05 工具与 Skills

本章节帮助你理解 OpenClaw 的工具系统（Tools）和技能系统（Skills），掌握如何扩展 Agent 能力。

## 🗺️ 三层架构全景

OpenClaw 工具与技能体系遵循**三层同心圆架构**：

```
🟠 Layer 1 Core（~8工具） ─ 原子操作：文件/执行/基础网络
🔵 Layer 2 Advanced（~17工具） ─ 进阶能力：浏览器/记忆/自动化
🟡 Layer 3 Knowledge（53+Skills） ─ 场景封装：办公/开发/安全/生活
```
