4.4.2 API 获取 ​

注册 Anthropic 账号 ​

步骤：
1. 访问 console.anthropic.com
2. 使用邮箱注册
3. 完成手机验证
4. 添加付款方式

创建 API Key ​

流程：
1. 登录 Claude 控制台
2. 点击「API Keys」
3. 点击「Create Key」
4. 命名并复制
5. 保存备用

充值方式 ​

支持的支付方式：
├── 信用卡 (Visa, Mastercard)
└── 美元账户

计费方式：
- 按 token 计费
- 无月费
- 可设置预算上限

json

    "claude-sonnet": {
      "provider": "anthropic",

      "max_tokens": 4096

    "claude-haiku": {

      "model": "claude-3-haiku-20240307",
      "api_key": "sk-ant-xxxxxxxxxxxxxxxx",

4.4.4 Claude 特有参数 ​

thinking (Claude 3.5 Sonnet) ​

thinking 参数控制思考模式：

"thinking": {
  "type": "enabled",
  "budget_tokens": 1024
}

- enabled: 启用思考模式
- disabled: 禁用思考模式
- budget_tokens: 思考预算

4.4.6 常见问题 ​

Q1: API Key 格式错误 ​

错误：invalid api key

Claude API Key 格式：
sk-ant-xxxxxxxxxxxxxxxx

注意：
- 不是 sk- 开头
- 是 sk-ant- 开头
- 检查复制是否完整

Q2: 速率限制 ​

错误：Rate limit exceeded

限制：
- 免费版：50 RPM
- API 版：根据套餐

解决：
1. 申请提高限制
2. 使用缓存
3. 批量处理

Q3: 上下文超限 ​

错误：context limit exceeded

解决：
1. 减少历史消息
2. 启用消息压缩
3. 使用摘要功能

Q4: 国内访问 ​

解决：
1. 使用 OpenRouter 转发
2. 使用代理服务
3. 使用国内平替模型
