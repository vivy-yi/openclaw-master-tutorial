# 9.5 LobsterBoard：仪表盘构建器

## 本节目标
- 了解 LobsterBoard 功能
- 掌握部署方法

---

## 5.1 工具简介

**LobsterBoard** 是一个仪表盘构建器，可以创建自定义的数据展示面板。

### 核心功能

- 自定义仪表盘
- 灵活的数据展示
- 实时数据更新

---

## 5.2 部署

```bash
# 克隆仓库
git clone https://github.com/Curbob/LobsterBoard.git
cd LobsterBoard

# 安装依赖
npm install

# 启动
npm run dev
```
