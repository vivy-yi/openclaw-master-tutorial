### Telegram 5 分钟极速上手

```bash
# 1. 向 @BotFather 申请 Bot，获得 Token
# 2. 在 openclaw.json 配置：
{
  "channels": {
    "telegram": {
      "enabled": true,
      "botToken": "你的token"
    }
  }
}
# 3. 重启
openclaw gateway restart
# 4. 给 Bot 发消息，应该有响应了
```

## 🚶 快速入门

**目标**：零基础到跑通第一个 Agent

### Step 1：安装（15分钟）

```bash
# macOS
brew install openclaw
openclaw install

# Linux
curl -fsSL https://get.openclaw.ai/install.sh | sh

# Docker
docker run -d -p 18789:18789 openclaw/openclaw
```

### Step 2：配置模型（10分钟）

```bash
# 用 DeepSeek（最便宜，效果好）
openclaw config set providers.deepseek.apiKey YOUR_KEY
openclaw config set agents.defaults.model deepseek/deepseek-chat

# 或用 OpenAI
openclaw config set providers.openai.apiKey YOUR_KEY
```

### Step 3：发出第一个指令（5分钟）

```bash
openclaw
> 你好，我是你的 OpenClaw 助手
> 帮我规划明天的日程
> 设置一个20分钟后提醒我喝水
```

### Step 4：接入 Telegram（10分钟）

见上方「渠道接入」。

## 📚 docs/ 和 chapters/ 是什么关系？

```
docs/          → 功能模块参考手册（按技术域分类）

architect-guide/ → 架构师深度文档（面向开发者/架构师）
```

**建议**：
- **用户** → 从 docs/ 找具体功能的用法
- **学习者** → 从 chapters/ 按顺序学习
- **开发者/架构师** → 从 architect-guide/ 深入原理
